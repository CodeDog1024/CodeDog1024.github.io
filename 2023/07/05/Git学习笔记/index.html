git reflog 查看历史记录
git log 查看日志

Git 分支

什么是分支

  • 在版本控制中，可以同时推进多个任务进行，每个任务都会有一个分支，程序员可以在除主分支以外的其他分支进行开发，从而不会因为发生事故而影响主分支的运行。

分支的操作

命令代码 命令功能
git branch 分支名 创建分支
git branch -v 查看分支
git checkout 分支名 切换分支
git merge 分支名 把指定分支合并到当前分支

远程仓库操作

命令代码 命令功能
git remote -v 查看所有远程仓库别名
git remote add 别名 远程仓库地址 为远程仓库起别名
git push 别名 分支 将本地分支推送到远程仓库
git clone 远程地址 将远程仓库的代码克隆到本地
git pull 别名 分支名 将远程仓库对于该分支的最新内容更新到本地库
