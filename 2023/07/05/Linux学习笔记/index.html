网络连接的三种模式

Linux目录结构

基本介绍

  • 根目录：“/”
  • 树状文件
  • 一切皆文件

常用目录

  • /bin：常用指令
  • /sbin：管理员使用的系统管理程序
  • /home：普通用户主目录
  • /root：管理员主目录
  • /etc：配置文件
  • /usr：应用程序文件
  • /boot：启动 Linux 的核心文件
  • /proc：虚拟目录，系统内存的映射
  • /srv：服务数据
  • /sys：
  • /temp：临时文件
  • /dev：设备文件
  • /mnt：外部文件存储设备
  • /opt：额外软件安装目录

  • 下载地址

  • Xshell：远程对主机进行操作

  • Xftp：远程对主机进行文件的上传和下载

Linux 用户命令

代码 作用
shutdow -h now 现在关机
shutdow -h 1 一分钟之后关机
shutdow -r now 现在重启
halt 关机，同上
reboot 重启，同上
sync 把内存中的数据同步到磁盘（关机重启前必须执行一次，防止数据丢失）
su - 用户名 切换用户
logout 退出当前账号登陆

Linux 用户管理命令(管理员模式下)

代码 作用
useradd 用户名 添加用户（自动在/homt文件夹下创建用户文件夹）
useradd -d 指定文件夹 用户名 添加用户（指定在/homt文件夹下的用户文件夹）
passwd 用户名 修改用户名
userdel 用户名 删除用户（但是会保留/home下的用户文件夹）
userdel -r 用户名 删除用户（不会保留/home下的用户文件夹）
id 用户名 查询用户信息
su - 用户名 切换用户（低权限切换高权限需要输入密码）
who am i 查看当前登录的用户

用户组

命令 功能
groupadd 组名 添加组
groupdel 组名 删除组
useradd -g 组名 用户名 添加用户时指定用户组
usermod -g 组名 用户名 切换用户组

用户和组的相关文件

  • /etc/passwd : 用户的配置文件，记录用户的各种信息
    • 用户名：口令：用户标识号：组标识号：注释行描述：主目录：登录Shell
  • /etc/shadow ：口令的配置文件
    • 登录名：加密口令：最后一次修改时间：最小时间间隔：最大时间间隔：警告时间：不活动时间：失效时间：标志
  • /etc/group ：组的配置文件，记录组的各种信息
    • 组名：口令：组标识号：组内用户列表（组内用户列表是隐藏的）

Linux 实用指令

运行级别

  1. 关机
  2. 单用户【找回丢失密码】
  3. 多用户状态没有网络服务
  4. 多用户状态有网络服务
  5. 系统未使用保留给用户
  6. 图形界面
  7. 系统重启

  • 常用级别是 3 和 5，也可以指定级别

  • init 运行级别 ：切换运行级别

  • systemctl get-default ：查看当前运行级别
