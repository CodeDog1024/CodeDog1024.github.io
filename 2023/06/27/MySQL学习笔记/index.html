MySQL 学习笔记

发表于2023-06-27|更新于2023-07-03

day 01

什么是数据库？什么是数据库管理系统？什么是 SQL ？

  1. 数据库：DataBase 简称 DB，按照一定格式存储数据的一些文件的组合。这些文件中存储了特定格式的数据
  2. 数据库管理系统：DataBaseManagement 简称 DBMS，用来管理数据库中的数据（增删改查）。常见数据库管理系统：MySQL、Oracle、MS SqlServer、DB2、sybase……
  3. SQL：结构化查询语句，程序员通过编写 SQL 语句，交给 DBMS 执行，一次来完成数据库中数据的增删改查。（在任何 DBMS 中通用 ）
  4. 三者关系；DBMS –执行–> SQL –操作–> DB

安装 MySQL 数据库管理系统

  1. 安装，选择经典版
  2. 配置 MySQL
  3. 端口：3306、字符集：utf-8、服务名称：MySQL、配置环境变量

完美卸载 MySQL

  1. 使用安装包移除 MySQL
  2. 删除目录（注意隐藏文件）

查看 MySQL 服务并启动

  1. 此电脑 –右键–> 管理 —-> 服务和应用程序 —-> MySQL
  2. 使用 cmd 命令 net start 服务名

登录 MySQL 及如何使用

  1. 登录命令 msyql -u root -p 密码

常用命令

命令（不区分大小写） 功能
mysql -u root -p 登录
exit 退出
select vision() 查看数据库版本
show databases 查看全部数据库
use 数据库名 使用指定数据库
select database 查看当前使用的哪个数据库
create database 数据库名 创建数据库
show tables 查看当前数据库全部表
desc 表名 查看表结构

数据库的基本单元

  1. 数据库的基本单元是表：table
  2. 一张表由行（row）和列（column）组成，每一行称记录，每一列称字段

SQL 语句的分类

  1. DQL：数据查询语言，查（select）
  2. DML：数据操作语言，增（insert）、删（delete）、改（update）
  3. DDL：数据定义语言，操作表的结构，如建表（create）、删除表（drop）、修改表（alter）……
  4. TCL：事务控制语言，事务提交（commit）、事务回滚（rollback）
  5. DCL：数据控制语言，授权（grant）、撤销权限（rovoke）……

  3. 给字段起别名

  5. 条件查询

    • 查询指定条件的数据
    sql 含义
    = 等于
    <>或者!= 不等于
    < 小于
    <= 小于等于
    > 大于
    >= 大于等于
    between 值1 and 值2 值在值1和值2之间的
    is null 为 null

    in 包含
    not 同 is 和 in使用，非的意思

  7. 数据处理函数（单行）

    函数 作用
    lower（字符串） 转小写
    upper（字符串） 转大写
    substr（字符串，起始位置，截取长度） 截取字符串
