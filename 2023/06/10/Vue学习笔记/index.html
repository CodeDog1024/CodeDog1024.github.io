六、Vue 指令

1、v-once

可以使用 v-once 来禁止数据被修改

2、v-html

当 data 中的数据为 html 标签语法的字符串时，可通过 v-html 讲字符串转为 html 标签语法

3、v-text

与 Mustache 语法功能相同，将 data 中的变量插入到 html 标签中，但是它会覆盖掉标签内的原有的数据

4、v-pre

不会解析Mustache语法内的变量，使其原封不动的显示

5、v-clock

在页面加载时，由于各种原因，Vue不能及时渲染数据，导致用户在页面直接看到 ,通过设置 v-clock 的样式（display: none），使得添加有 v-clock 的标签，再Vue未渲染数据之前不会显示，当Vue渲染数据之后，会自动删除v-clock,从而显示正确的数据

6、v-bind

将 data 内的变量绑定到 html 标签的属性当中，比如a标签的href属性，img标签的src属性

七、计算属性 computed

1、计算属性的使用

计算属性是一个函数，它的返回值会被赋值给它的变量，它的返回值可以是任意类型，但是它的返回值必须是唯一的。

计算属性的返回值会被缓存，并且只有当它的依赖发生了变化，才会重新计算。

计算属性的依赖可以是多个，但是它们的值必须是唯一的。

计算属性 案例

计算属性的复杂操作 案例

2、set 和 get

set 函数用来设置属性的值，get 函数用来获取属性的值

3、computed 和 methods 的区别

methods 在每一次调用的时候都会执行

computed 在第一次调用的时候才会执行，之后每次调用都会直接返回缓存的值

1、let 和 var 的区别

let 变量只能在定义的语句块内有效，var 变量可以在定义的语句块外有效 作用域 案例

2、const 和 let 的区别

const 变量是常量，它的值不能被改变，它的值不可以被重新赋值

let 变量是变量，它的值可以被改变，它的值可以被重新赋值

九、事件监听 v-on

1、事件监听的使用

事件监听的使用方式，可以使用 v-on 来监听事件，也可以使用 @ 来监听事件。监听事件总结

2、传参的使用

不需要传参时，调用方法时可以省略 ()

需要传参时，不可省略 ()

当方法中需要接收参数，但是调用时并未传参，则参数默认为 事件对象

当参数和事件对象同时作为参数传递时，调用时应用 $event 作为事件对象的参数

3、v-on 的修饰符

stop 停止冒泡

prevent 阻止默认行为

按键修饰符，在监听键盘按下时（keyup），可以在 keyup 后添加指定按键来监听,
比如 keyup.enter

once 只监听一次

十、条件判断

1、v-if 的使用

可以使用 v-if 来判断是否显示某个元素，如果条件为 true，则显示，否则不显示

条件判断 案例

登录案例 案例

2、v-show 的使用

可以使用 v-show 来判断是否显示某个元素，如果条件为 true，则显示，否则不显示

3、v-show 和 v-if 的区别

带有 v-if 的标签，不会存在与 dom 中

带有 v-show 的标签，会存在与 dom 中，只是添加了 display: none; 样式

频繁切换使用 v-show 比较好，因为 v-show 只会添加或删除样式，不会改变 dom 的结构

十一、v-for

1、v-for 的使用

可以使用 v-for 来遍历数组，或者对象的属性，并且可以使用 key 来指定每个元素的 id

2、key 的使用

可以使用 key 来指定每个元素的 id，这样可以在每次渲染时，都会检查是否有相同 id 的元素，如果有，则会自动删除旧的元素，并且新的元素会替换旧的元素，这样可以避免重复元素的出现

如果 key 不存在，则会自动生成 id，这样可以避免重复元素的出现

key 的作用主要是为了高效的更新dom

3、数组操作

响应式方法：push、pop、shift、unshift、splice、sort、reverse、set

4、数组遍历

for(let i = 0; i < this.books.length; i++) {…}

for(let i in this.books) {…}

for(let i of this.books) {…}

this.books.forEach(function(item, index) {…}

购物车案例 案例

5、高阶函数

可以使用高阶函数来给数组添加方法，比如 map、filter、reduce、find、findIndex、some、every 等方法

filter 方法：返回一个新数组，包含通过测试的所有元素,例如：students.filter(function(stu)) { return stu.age > 18 })

map 方法：返回一个新数组，包含映射后的所有元素,例如：students.map(function(stu)) { return stu.age + 1})

reduce 方法：返回一个值，即使用给定函数计算数组中所有元素的值的和
,例如：students.reduce(function(total, stu)) { return total + stu.age })

十二、v-model

1、v-model 的使用

可以使用 v-model 来让表单元素和 vue 实例的属性进行绑定，这样就可以实现表单的数据绑定

2、修饰符

lazy 模式：当输入值变化时，只会触发一次输入事件，而不会触发多次输入事件

number 模式：只能输入数字

trim 模式：去除输入值的首尾空格

十八、vue-router

1、安装

npm install vue-router –save

2、使用

import VueRouter from ‘vue-router’

3、配置

index.js

4、组件

路由跳转 router-link

路由显示 router-view

5、动态路由

在 routes 中定义路由映射时，在 path 中使用动态路径，例如：/user/:id，
在进行页面跳转时，需要传递 id 参数，例如：router.push(‘/user/1’)
而后也可通过 this.$route.params 获取路由参数，例如：this.$route.params.id

6、懒加载

在配置路径映射路径时，配置 commponent 时，使用 ()=>import(‘@/components/xxx’)

7、嵌套路由

在配置路径映射时，通过children来配置子组件路由，并在该组件使用router-view显示子组件

8、参数传递

在 router-view 中使用 to 属性是 传递参数为一个对象，例如：to={ path: ‘/user/:id’, query: { id: 1 } }

9、route 和 router 的区别

router：是一个对象，包含 routes、mode、base、linkActiveClass 等属性，
其中 routes 是一个数组，包含 route 对象，其中 route 对象也是一个对象，包含 path、name、component、children 等属性。

10、导航守卫

全局守卫：在index.js中，使用 router.beforeEach 方法，

单独守卫：在组件中，使用 beforeEnter 属性，

在路由配置中，可以使用 beforeEach 方法，它是一个函数，
当路由进入时，会调用该函数，需要传递三个函数，分别是：to、from、next，
在该函数中，可以调用 next 方法，来控制路由跳转，
如果不调用 next 方法，则不会跳转，可以使用 next(false) 来阻止跳转。

11、keep-alive

keep-alive 时 Vue 内置的组件，可以保留组件的状态，避免重新渲染
