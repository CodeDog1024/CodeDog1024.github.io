HTML 标签的介绍

HTML 标签是 HTML 文档的构建块，用于定义网页的元素。

标签按照格式分为：单标签和双标签。

单标签：如<br />、<hr />、<img src="xxx" /> 这些由一个标签开始，没有结束标签。
双标签：如<p></p>、<div></div>、<h1></h1> 等，由开始标签和结束标签组成。

个别标签还拥有属性，如 <img src="xxx" /> 中 src 属性用于指定图片的源文件。

个别标签还拥有事件，如 <button onclick="xxx()"></button> 中 onclick 属性用于指定按钮点击时执行的 JavaScript 代码。

注意事项

  1. 双标签内可以嵌套其他标签，但是不可以交叉嵌套：

    • 正确：<div><p>文字</p></div>
    • 错误：<div><p>文字</div></p>

  2. 双标签必须关闭：

    • 正确：<div>文字</div>
    • 错误：<div>文字

  3. 单标签也需要关闭：

    • 正确：<br />
    • 错误：<br>

  4. 标签中的属性必须使用引号括起来：

    • 正确：<img src="xxx" />
    • 错误：<img src=xxx />

  5. 注释不能嵌套：

    • 正确：<!-- 注释 -->
    • 错误：<!-- <!-- 注释 --> -->
