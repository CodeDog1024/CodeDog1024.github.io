HTML 标签的介绍

HTML 标签是 HTML 文档的构建块，用于定义网页的元素。

标签按照格式分为：单标签和双标签。

单标签：如<br />、<hr />、<img src="xxx" /> 这些由一个标签开始，没有结束标签。
双标签：如<p></p>、<div></div>、<h1></h1> 等，由开始标签和结束标签组成。

个别标签还拥有属性，如 <img src="xxx" /> 中 src 属性用于指定图片的源文件。

个别标签还拥有事件，如 <button onclick="xxx()"></button> 中 onclick 属性用于指定按钮点击时执行的 JavaScript 代码。

注意事项

  1. 双标签内可以嵌套其他标签，但是不可以交叉嵌套：

    • 正确：<div><p>文字</p></div>
    • 错误：<div><p>文字</div></p>

  2. 双标签必须关闭：

    • 正确：<div>文字</div>
    • 错误：<div>文字

  3. 单标签也需要关闭：

    • 正确：<br />
    • 错误：<br>

  4. 标签中的属性必须使用引号括起来：

    • 正确：<img src="xxx" />
    • 错误：<img src=xxx />

  5. 注释不能嵌套：

    • 正确：<!-- 注释 -->
    • 错误：<!-- <!-- 注释 --> -->

特殊字符

在 HTML 中，< 和 > 是具有特殊含义的字符，我们无法直接将其在浏览器中显示。此时就需要使用实体来替换。

字符 实体
< &lt;
> &gt;
& &amp;
“ &quot;
‘ &apos;
半角空格 &nbsp;
全角空格 &emsp;

标题标签

标题标签是 h1 ~ h6，从大到小排列。

超链接标签

在网页中，通过点击超链接可以跳转到其他页面。超链接标签为 a，属性有：href、target、title。

列表标签

列表标签分为有序列表和无序列表。有序列表为 ol，无序列表为 ul，其中的每一项都是 li 标签。

img 标签

img 标签用于插入图片。属性有：src、alt、width、height。

表格标签

表格标签为 table，其中分为 thead、tbody、tfoot，分别为表头、表体、表尾。其中又可以包含tr（行）、th（表头单元格）、td（数据单元格）。

跨行跨列表格

在一些情况下，我们需要将多个单元格合并为一个单元格。这时，我们可以使用 colspan 和 rowspan 属性。

colspan 属性用于合并列，rowspan 属性用于合并行。

iframe 标签

iframe 标签用于在当前页面中嵌入 another page。属性有：src、width、height。

表单标签

表单标签为 form，属性有：action、method、enctype。

表单标签用于创建一个表单，可以将表单标签中的值提交给服务器。

表单标签中包含的表单标签有：input、textarea、select、option、button、label、fieldset、legend 等。

其中 input 标签用于创建输入框，可以根据属性 type 来设置输入框的类型。
