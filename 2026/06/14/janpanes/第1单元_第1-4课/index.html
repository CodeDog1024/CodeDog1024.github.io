第1单元 语法整理（第1-4课）

第1课 李さんは 中国人です

语法1：～は～です（肯定句）

解析：

  • 「は」是提示助词，读作「wa」，用于提示主题
  • 「です」是断定助动词，表示礼貌的肯定判断
  • 句型：主题 + は + 名词 + です（～是～）

例句：

  • わたしは 王です。/ 我是小王。
  • 李さんは 中国人です。/ 小李是中国人。
  • 田中さんは 日本人です。/ 田中是日本人。
  • これ は 本です。/ 这是书。

语法2：～は～ではありません（否定句）

解析：

  • 「ではありません」是「です」的否定形式
  • 「じゃ」是「では」的口语缩略形式
  • 句型：主题 + は + 名词 + ではありません（～不是～）

例句：

  • わたしは 学生じゃ ありません。/ 我不是学生。
  • 田中さんは 中国人では ありません。/ 田中不是中国人。
  • これは 本じゃ ありません。/ 这不是书。

语法3：～は～ですか（疑问句）

解析：

  • 「か」是疑问助词，加在句末构成疑问句
  • 「ですか」是礼貌的疑问形式
  • 回答时用「はい」（是）或「いいえ」（不是）

例句：

  • あの方は どなたですか。/ 那位是谁？
  • 田中さんですか。/ 是田中吗？
  • — 田中さんですか。/ — はい、田中です。/ 是田中吗？— 是的，是田中。
  • — 王さんですか。/ — いいえ、王じゃ ありません。/ 是小王吗？— 不是，不是小王。

语法4：疑问词

解析：

  • だれ（谁）、なに（什么）、どこ（哪里）、いつ（什么时候）、いくら（多少钱）

例句：

  • それは 何ですか。/ 那是什么？
  • あの方は だれですか。/ 那个人是谁？
  • 学校は どこですか。/ 学校在哪里？

语法5：～の～（所属关系）

解析：

  • 「の」是连体助词，连接两个名词，表示所属、内容等关系
  • A の B = B of A / A的B

例句：

  • わたしは 会社の 社長です。/ 我是公司的总经理。
  • 田中さんの かさです。/ 田中的伞。
  • 日本の 会社です。/ 日本的公司。
  • 中国の 北京は 首都です。/ 中国的北京是首都。

语法6：指示代词「こそあ」

解析：

  • これ（这个，靠近说话人）
  • それ（那个，靠近听话人）
  • あれ（那个，远离双方）
  • どの + 名词（哪一个）

例句：

  • これは 何ですか。/ 这是什么？
  • それは 田中さんの かさですか。/ 那是田中的伞吗？
  • あれは だれの カメラですか。/ 那是谁的照相机？

语法7：连体词「この/その/あの」

解析：

  • この + 名词 = 这个～
  • その + 名词 = 那个～
  • あの + 名词 = 那个～（远方的）
  • どの + 名词 = 哪个～

例句：

  • この 本は わたしの です。/ 这本书是我的。
  • その 時計は だれの ですか。/ 那块表是谁的？
  • あの 人は 誰ですか。/ 那个人是谁？

第2课 これは 本です

语法1：～は～も～（也）

解析：

  • 「も」是提示助词，表示”也”
  • 替换「は」使用，表示主题的并列

例句：

  • わたしは 日本語も 英語も わかります。/ 我既懂日语也懂英语。
  • ビールも 飲みます。/ 也喝啤酒。
  • コーヒーも 好きです。/ 也喜欢咖啡。

语法2：～は～ではありません（强调否定）

解析：

  • 「も～ではありません」也都不～
  • 用于完全否定

例句：

  • 酒も たばこも ません。/ 烟酒都不沾。
  • 中国語も 英語も わかりません。/ 中文和英文都不懂。

第3课 ここは デパートです

语法1：场所代词

解析：

  • ここ（这里，靠近说话人）
  • そこ（那里，靠近听话人）
  • あそこ（那里，远离双方）
  • どこ（哪里）

例句：

  • ここは 学校です。/ 这里是学校。
  • そちらは お手洗いですか。/ 那边是洗手间吗？
  • あそこは デパートです。/ 那里是百货商店。

语法2：～は～の上/下/前/後ろ/右/左にあります/います

解析：

  • 表示某物/某人在某处
  • に：表示存在的场所
  • あります：无生命物体的存在
  • います：有生命物体的存在

方位词：

  • 上（うえ）：上面
  • 下（した）：下面
  • 前（まえ）：前面
  • 後ろ（うしろ）：后面
  • 右（みぎ）：右边
  • 左（ひだり）：左边
  • 中（なか）：里面
  • 外（そと）：外面
  • 近い（ちかい）附近
  • 隣（となり）：旁边

例句：

  • 机の 上に 本が あります。/ 桌子上面有书。
  • 椅子の 下に 猫が います。/ 椅子下面有猫。
  • デパートの 前に バスが あります。/ 百货商店前面有公共汽车。
  • あの人だれですか。/ あの人だれですか。/ 那个人是谁？
  • 王さんも 同僚です。/ 小王也是同事。

语法3：～か～（选择疑问句）

解析：

  • 用于列举选项供对方选择
  • 句型：～か～ですか（是～还是～？）

例句：

  • これと どれが 好きですか。/ と和どれ 哪个更喜欢？
  • コーヒーと 紅茶と、どちらが 好きですか。/ 咖啡和红茶，喜欢哪个？
  • お茶は どうですか。/ 茶怎么样？
  • すみません、田中さんの カバンは どこですか。/ 请问田中的包在哪里？

语法4：～から～まで（从～到～）

解析：

  • 「から」表示起点（从）
  • 「まで」表示终点（到）

例句：

  • 午前9時から 5時まで 働きます。/ 从上午9点工作到5点。
  • 北京から 上海まで 電車で 8時間です。/ 从北京坐电车到上海要8小时。

语法5：～に～があります/います

解析：

  • 表示某处存在某物/某人
  • 场所 + に + 存在物 + があります/います

例句：

  • 部屋に テーブルが あります。/ 房间里有桌子。
  • 公園に 子供が います。/ 公园里有孩子。
  • 机の 上に 何が ありますか。/ 桌子上有什么？
  • 教室に だれが いますか。/ 教室里有谁？

第4课 部屋に 机と 椅子が あります

语法1：～と～（和）

解析：

  • 「と」是并列助词，表示”和”
  • 用于连接名词，列举全部

例句：

  • わたしは パンと ジュースを 食べます。/ 我吃面包和果汁。
  • テーブルの 上に 本と ノートが あります。/ 桌子上有书和笔记本。

语法2：～か～（或者）

解析：

  • 「か」是并列助词，表示”或者”
  • 用于列举部分选项

例句：

  • お茶か コーヒーを 飲みます。/ 喝茶或咖啡。
  • 土曜日か 日曜日に 行きます。/ 星期六或星期天去。

语法3：～も～も～（都）

解析：

  • 「も～も」表示”都”，全部肯定
  • 名词1 + も + 名词2 + も + 相同的谓语

例句：

  • ビールも ウイスキーも 飲みます。/ 啤酒和威士忌都喝。
  • 東京も 大阪も 行きました。/ 东京和大阪都去了。

语法4：数量词 + に + ～ます（分配基准）

解析：

  • 表示分配的基准
  • 数量词 + に + 动词

例句：

  • 3人に 1冊ずつ あげます。/ 每3人给1本。
  • 1週間に 3回 テニスを します。/ 每周打3次网球。

语法5：～は～が好きです/上手です

解析：

  • 表示好恶、擅长不擅长时，对象用「が」
  • 好き（すき）：喜欢
  • 嫌い（きらい）：讨厌
  • 上手（じょうず）：擅长
  • 下手（へた）：不擅长

例句：

  • わたしは 野菜が 好きです。/ 我喜欢蔬菜。
  • ミラーさんは 歌が 上手です。/ 米勒擅长唱歌。
  • わたしは 料理が 下手です。/ 我不擅长做饭。

语法6：～から～まで（期间）

解析：

  • 表示时间的起点和终点
  • ～から～まで：从～到～

例句：

  • 月曜日から 金曜日まで 働きます。/ 从周一到周五工作。
  • 9月から 11月まで 雨が 多いです。/ 从9月到11月雨很多。

相关语法比较

「は」vs「が」

は が
功能 提示主题 强调主语/对象
新信息 旧信息（已知） 新信息（未知）
例句 わたしは 学生です。/ 我是学生。 誰が 学生ですか。/ 谁是学生？

「あります」vs「います」

あります います
对象 无生命的事物 有生命的人或动物
例句 本が あります。/ 有书。 猫が います。/ 有猫。

「の」vs「の」

用法 例句 含义
所属 田中さんの 本 田中的书
内容 日本語の 本 日语书
同位 友達の 田中さん 朋友田中
材质 木の テーブル 木制桌子
