第7单元 语法整理（第25-28课）

第25课 母の 日に 何を 買いましたか

语法1：～を ～ました（做了～）

解析：

  • 动词ます形去掉「ます」+ ました
  • 表示过去完成了某个动作

例句：

  • 母の 日に 何を 買いましたか。/ 母亲节买了什么？
  • 昨日 新しい 本を 買いました。/ 昨天买了新书。
  • 朝ごはんを 食べました。/ 吃了早饭。

语法2：～に～を あげました（给了～）

解析：

  • に：给的对象
  • を：东西
  • あげました：给了

例句：

  • 母に 花を あげました。/ 给妈妈花了。
  • 友達に プレゼントを あげました。/ 给朋友礼物了。
  • 先生に お手紙を あげました。/ 给老师信了。

语法3：～に～を もらいました（收到了～）

解析：

  • に：给的对象
  • を：东西
  • もらいました：收到了

例句：

  • 母に 花を もらいました。/ 从妈妈那里收到了花。
  • 友達に プレゼントを もらいました。/ 从朋友那里收到了礼物。
  • 先生に おもちゃを もらいました。/ 从老师那里收到了玩具。

语法4：～に～を くれました（给了～）

解析：

  • に：给的对象
  • を：东西
  • くれました：（别人）给了我

例句：

  • 母に 花を くれました。/ 妈妈给我花了。
  • 友達に プレゼントを くれました。/ 朋友给我礼物了。

第26课 写真を 撮りました

语法1：～を～ました（做了～）

解析：

  • 动词ます形去掉「ます」+ ました

例句：

  • 写真を 撮りました。/ 拍了照片。
  • 花を 撮りました。/ 拍了花。

语法2：～て くださいました（为我做了～）

解析：

  • 动词て形 + くださいました
  • 表示别人为自己做了某事（感谢）

例句：

  • 花を 撮って くださいました。/ 为我拍了花。
  • 写真を 撮って くださいました。/ 为我拍了照片。

语法3：～て あげました（为～做了）

解析：

  • 动词て形 + あげました
  • 表示自己为别人做了某事

例句：

  • 友達に 写真を 撮って あげました。/ 为朋友拍了照片。
  • 子供に おもちゃを 買って あげました。/ 为孩子买了玩具。

语法4：～て もらいました（请～做了）

解析：

  • 动词て形 + もらいました
  • 表示请别人为自己做了某事

例句：

  • 友達に 写真を 撮って もらいました。/ 请朋友拍了照片。
  • 先生に 英語を 教えて もらいました。/ 请老师教了英语。

第27课 旅行の 予定は ありません

语法1：～予定です（预定～）

解析：

  • 动词基本形 / ない形 + 予定です
  • 表示计划、预定

例句：

  • 旅行の 予定は ありません。/ 没有旅行的计划。
  • 来週 日本に 行く予定です。/ 下周预定去日本。
  • 今晩 何も しない予定です。/ 今晚预定什么都不做。

语法2：～つもりです（打算～）

解析：

  • 动词基本形 / ない形 + つもりです
  • 表示打算、意图

例句：

  • 日本に 行くつもりです。/ 打算去日本。
  • 週末 何も しないつもりです。/ 周末打算什么都不做。

语法3：～ます（肯定）

解析：

  • 动词肯定形式

例句：

  • 何を 食べますか。/ 吃什么？
  • 何を 飲みますか。/ 喝什么？

语法4：～ません（否定）

解析：

  • 动词否定形式

例句：

  • 何も 食べません。/ 什么都不吃。
  • 酒を 飲みません。/ 不喝酒。

语法5：～ました（过去肯定）

解析：

  • 动词过去肯定形式

例句：

  • 昨日 买いました。/ 昨天买了。
  • 先週 行きました。/ 上周去了。

语法6：～ませんでした（过去否定）

解析：

  • 动词过去否定形式

例句：

  • 昨日 何も 食べませんでした。/ 昨天什么都没吃。
  • 何も 買いませんでした。/ 什么都没买。

第28课 何を しますか

语法1：～を します（做～）

解析：

  • 动词ます形去掉「ます」+ します
  • 表示做某事

例句：

  • 何を しますか。/ 做什么？
  • 運動を します。/ 做运动。
  • 仕事を します。/ 工作。
  • 予習を します。/ 预习。
  • 復習を します。/ 复习。

语法2：～で します（决定做～）

解析：

  • 名词 + で + します
  • 表示决定

例句：

  • コーヒーで します。/ 决定要咖啡。
  • パンで します。/ 决定要面包。
  • これで します。/ 决定要这个。

语法3：～が いいです（～比较好）

解析：

  • 动词た形 + が いいです
  • 表示建议

例句：

  • 早く 起きた が いいです。/ 最好早点起床。
  • 早く 帰った が いいです。/ 最好早点回去。

语法4：～た方が いいです（最好～）

解析：

  • 动词た形 + 方（ほう）が いいです
  • 表示建议

例句：

  • 早く 起きた 方が いいです。/ 最好早点起床。
  • 早く 帰った 方が いいです。/ 最好早点回去。

语法5：～ない方が いいです（最好不要～）

解析：

  • 动词ない形 + 方（ほう）が いいです
  • 表示否定建议

例句：

  • あまり 食べない 方が いいです。/ 最好不要吃太多。
  • 遅く 帰らない 方が いいです。/ 最好不要太晚回去。

相关语法比较

「あげます」vs「もらいます」vs「くれます」

动词 方向 例句
あげます 我 → 别人 友達に プレゼントを あげました。
もらいます 别人 → 我 友達に プレゼントを もらいました。
くれます 别人 → 我 友達に プレゼントを くれました。

「～つもりです」vs「～予定です」

形式 含义 例句
つもりです 打算（主观意图） 日本に 行くつもりです。
予定です 预定（客观安排） 来週 会議がある予定です。

建议表达

形式 含义 例句
～た方が いいです 最好～ 早く 起きた方 が いいです。
～ない方が いいです 最好不要～ あまり 食べない方 が いいです。
