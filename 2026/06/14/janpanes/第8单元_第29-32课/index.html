第8单元 语法整理（第29-32课）

第29课 午前11時半に 会社を 出ます

语法1：～を 出ます（离开～）

解析：

  • 场所 + を + 出ます
  • 「を」表示离开的场所

例句：

  • 会社を 出ます。/ 离开公司。
  • 家を 出ます。/ 离开家。
  • 学校を 出ます。/ 离开学校。
  • 部屋を 出ます。/ 离开房间。

语法2：～に 行きます/来ます（目的地）

解析：

  • 场所 + に + 行きます/来ます
  • 「に」表示目的地

例句：

  • 学校に 行きます。/ 去学校。
  • 会社に 来ます。/ 来公司。
  • 日本に 行きます。/ 去日本。

语法3：～へ 行きます/来ます（方向）

解析：

  • 场所 + へ + 行きます/来ます
  • 「へ」表示方向

例句：

  • 学校へ 行きます。/ 去学校。
  • 日本へ 行きます。/ 去日本。

语法4：～から～まで（时间范围）

解析：

  • 从～到～

例句：

  • 午前11時半に 会社を 出ます。/ 上午11点半离开公司。
  • 9時から 5時まで 働きます。/ 从9点工作到5点。

第30课 電車が 止まりました

语法1：～が～ました（发生了某事）

解析：

  • 主语 + が + 动词ました
  • 表示发生了某事

例句：

  • 電車が 止まりました。/ 电车停了。
  • 雨が 降りました。/ 下雨了。
  • 風が 吹きました。/ 刮风了。
  • 花が 咲きました。/ 花开了。

语法2：～に～ました（在某处发生了某事）

解析：

  • 场所 + に + 动词ました

例句：

  • 駅に 電車が 止まりました。/ 电车在车站停了。
  • 公園に 花が 咲きました。/ 公园里花开了。

语法3：～が～ています（状态）

解析：

  • 动词て形 + います
  • 表示动作结果的状态

例句：

  • 電車が 止まっています。/ 电车停着。
  • 花が 咲いています。/ 花开着。
  • 窓が 開いています。/ 窗开着。

语法4：～に～ています（状态持续）

解析：

  • 场所 + に + 动词て形 + います

例句：

  • 駅に 電車が 止まっています。/ 电车在车站停着。
  • 公園に 花が 咲いています。/ 公园里花开着。

第31课 重い 持ち物が ありますか

语法1：～が ありますか（有～吗）

解析：

  • 名词 + が ありますか

例句：

  • 重い 持ち物が ありますか。/ 有重的随身物品吗？
  • 日本語の 本が ありますか。/ 有日语书吗？
  • テレビが ありますか。/ 有电视吗？

语法2：～を 持っていますか（拿着～吗）

解析：

  • 动词て形 + います
  • 表示状态

例句：

  • 何を 持っていますか。/ 拿着什么？
  • 日本語の 本を 持っています。/ 拿着日语书。
  • カメラを 持っています。/ 拿着照相机。

语法3：～に～があります/います（某处有某物）

解析：

  • 场所 + に + 存在物 + が あります/います

例句：

  • 部屋に テーブルが あります。/ 房间里有桌子。
  • 公園に 猫が います。/ 公园里有猫。
  • 冷蔵庫に 何が ありますか。/ 冰箱里有什么？

语法4：～は～にあります/います（某物在某处）

解析：

  • 存在物 + は + 场所 + に あります/います

例句：

  • 本は 机の 上に あります。/ 书在桌子上。
  • 猫は 椅子の 下に います。/ 猫在椅子下面。

第32课 すみません、お手洗いは どこですか

语法1：疑问词 + ですか（～在哪里）

解析：

  • 疑问词 + ですか

例句：

  • お手洗いは どこですか。/ 洗手间在哪里？
  • 駅は どこですか。/ 车站在哪里？
  • あの人だれですか。/ 那个人是谁？
  • これは 何ですか。/ 这是什么？

语法2：～は どこですか（～在哪里）

解析：

  • 名词 + は どこですか

例句：

  • お手洗いは どこですか。/ 洗手间在哪里？
  • 駅は どこですか。/ 车站在哪里？
  • 学校は どこですか。/ 学校在哪里？

语法3：～は いくらですか（～多少钱）

解析：

  • 名词 + は いくらですか

例句：

  • この 本は いくらですか。/ 这本书多少钱？
  • カメラは いくらですか。/ 照相机多少钱？

语法4：～は いつですか（～什么时候）

解析：

  • 名词 + は いつですか

例句：

  • 試験は いつですか。/ 考试什么时候？
  • 誕生日は いつですか。/ 生日什么时候？

语法5：～は だれですか（～是谁）

解析：

  • 名词 + は だれですか

例句：

  • あの人だれですか。/ 那个人是谁？
  • 先生は だれですか。/ 老师是谁？

相关语法比较

「を」vs「に」（离开/到达）

用法 例句 说明
を（离开） 家を 出ます。 离开家
に（到达） 学校に 行きます。 到达学校

「あります」vs「います」

あります います
对象 无生命的事物 有生命的人或动物
例句 本が あります。 猫が います。

疑问词总结

疑问词 含义 例句
どこ 哪里 お手洗いは どこですか。
いくら 多少钱 この 本は いくらですか。
いつ 什么时候 試験は いつですか。
だれ 谁 あの人だれですか。
何 什么 これは 何ですか。
