JSP

JSP 是 Java Server Page 的简称，是一种动态网页技术。

早期的 HTML 页面只能静态显示数据，因此想要更新数据，就只能再后端通过字符串绘制 HTML 页面。然后传给前端，前端再进行展示。

JSP 允许在 HTML 中嵌入 Java 代码，从而实现动态网页。从而让 HTML 中的数据实现动态更新的效果。

JSP 本质

JSP 本质是一个 Java 类，通过编译成 .class 文件，再运行。该类继承了 HttpJspBase 类，而 HttpJspBase 类继承了 HttpServlet 类，所以 JSP 本质是 HttpServlet 类的一个子类。再该类的内部其实也是通过 resp 将 HTML 拼接成字符串返回给前端。只不过不用我们自己拼接字符串了，JSP 会自动完成拼接。

语法

Page 指令

Page 指令用于指定 JSP 页面的属性。

  1. language: 指定 JSP 页面的脚本语言。
  2. contentType: 指定 JSP 页面的内容类型。
  3. pageEncoding: 指定 JSP 页面的编码。
  4. import: 指定导入的类。
  5. autoFlush: 指定是否自动刷新。
  6. buffer: 指定缓冲区大小。
  7. errorPage: 错误页面。
  8. isErrorPage: 是否是错误页面。
  9. session: 是否使用 session。
  10. extends: 指定继承的父类。
