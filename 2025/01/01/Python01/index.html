初识 Python

Python 的起源

  • Python 的历史可以追溯到 1989 年，由 Guido van Rossum 创建，并成为第一个支持动态类型和面向对象编程的语言。
  • 1991 年，第一个 Python 解释器诞生

为什么选择 Python

  • 优雅
    • 简单
    • 易学
    • 开发效率高
  • 适用面广
    • 自动化脚本
    • 自动化办公
    • WEB 应用
    • 数据计算
    • AI 程序
    • 大数据
    • 计算机二级

什么是编程语言

什么是编程语言

  • 语言，即沟通交流的方式，编程语言就是人与计算机之间的沟通方式，而解释器是一个翻译器，将编程语言翻译成计算机可以理解的语言。

Python 安装

Python 安装（Windows）

  • Python 安装包下载地址
  • 注意：Add Python 3.10 to PATH 一定要打勾
  • 安装完成后，在 cmd 中输入 python，如果出现 Python 版本信息，则安装成功。

Python 安装（Linux）

  • 安装命令（Ubuntu）：sudo apt install python3
  • 注意：不同系统的 Linux 安装命令不同，请自行查询。

第一个 Python 程序

  • 注意：任何符号，如双引号，括号等，都需要是英文状态下的符号

小练习

常见问题

  • 打开命令提示符：win + r，输入 cmd，按下 Enter。或者点击 开始，在搜索框中输入 cmd 或 命令提示符，点击 打开。
  • 在命令提示符中输入 python 时提示 'python' 不是内部命令，也不是外部命令，也不是可运行的程序，是因为在安装 Pthon 时没有勾选 Add Python 3.10 to PATH，卸载重新安装，再勾选 Add Python 3.10 to PATH，即可解决该问题。
  • 指定代码时，提示 SyntaxError: invalid character 'XXX'，是因为该字符为非法字符，如 中文，特殊符号等，需要删除或修改为英文符号。

Python 解释器

Python 解释器

  • 计算机只认识二进制，Python 解释器的功能就是把 Python 代码翻译成计算机可以理解的语言，然后执行。安装 Pyython 的本质就是安装 Python 解释器。
  • 执行 Python 代码的方式有两种：一种时直接在 Python 解释器中编写代码然后执行。另一种是创建一个 Python 文件，在文件中编写 Python 代码，然后通过 Python 解释器执行该文件。
