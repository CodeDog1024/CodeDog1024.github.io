查找

基本查找

基本查找即顺序遍历查找，通过遍历全部元素，找到目标元素。时间复杂度为 O(n)，空间复度为 O(1)。

二分查找

二分查找，通过将数组划分为多个块，对每个块进行二分查找，找到目标元素。时间复杂度为 O(log n)，空间复杂度为 O(1)。二分查找要求数组是有序的。

插值查找

插值查找，通过计算目标元素在数组中的位置，然后通过二分查找找到目标元素。时间复杂度为 O(log log n)，空间复杂度为 O(1)。插值查找要求数组是有序的。

斐波那契查找

斐波那契查找，通过计算目标元素在数组中的位置，然后通过二分查找找到目标元素。时间复杂度为 O(log n)，空间复杂度为 O(1)。斐波那契查找要求数组是有序的。

分块查找

分块查找，通过将数组划分为多个块，块数一般为元素个数的平方根，要求前一块的最大值小于后一块的最小值，根据目标元素在哪个块中，然后在该块中进行顺序查找。时间复杂度为 O(√n)，空间复杂度为 O(1)。

排序

冒泡排序

冒泡排序，通过两两比较相邻元素，将最大（最小）元素放到数组末尾（首），然后重复上述过程，直到数组有序。时间复杂度为 O(n^2)，空间复杂度为 O(1)。

选择排序

选择排序，通过找到数组中的最小（最大）元素，将其放到数组的起始位置，然后重复上述过程，直到数组有序。时间复杂度为 O(n^2)，空间复杂度为 O(1)。

插入排序

插入排序，通过将数组划分为已排序和未排序两部分，将未排序部分的元素插入到已排序部分的正确位置，直到数组有序。时间复杂度为 O(n^2)，空间复杂度为 O(1)。

递归

递归，通过将问题分解为更小的子问题，直到子问题可以解决，然后通过组合子问题的解得到原问题的解。时间复杂度为 O(n)，空间复杂度为 O(n)。

快速排序

快速排序，通过将数组分为两个子数组，一个子数组中的元素都小于等于 pivot，另一个子数组中的元素都大于 pivot，然后递归对两个子数组进行排序。时间复杂度为 O(nlogn)，空间复杂度为 O(logn)。

Lambda

Lambda 表达式是 Java 8 引入的一种新特性，它是一种简洁的表示函数式接口的方式。函数式接口是指只包含一个抽象方法的接口。可以在接口上使用 @FunctionalInterface 注解来标记该接口是函数式接口。

Lambda 代码示例

注意事项

  1. Lambda 只能用于接口，不能用于类。
  2. Lambda 只能用于函数式接口。即该接口中只能有一个抽象方法。
  3. Lambda 语法格式：(参数列表) -> {方法体}
  4. 参数类型可以省略，编译器会自动推导出参数类型。
  5. 如果方法参数列表只有一个参数，可以省略小括号。
  6. 如果方法体只有一行代码，可以省略大括号和 return 关键字。
