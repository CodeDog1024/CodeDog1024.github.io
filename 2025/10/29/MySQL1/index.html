概述

数据库：数据库就是一个仓库，将数据进行有组织的存储管理。

MySQL: 是一个关系型数据库管理系统（Relational Database Management System，RDBMS），MySQL 并不是数据库。

SQL：SQL是Structured Query Language的缩写，是一种数据库查询和程序设计语言。我们需要通过编写 SQL 语句来操作数据库。

启动、停止

  1. 启动命令：net start 服务名
  2. 停止命令：net stop 服务名

连接

  1. 使用 MySQL 提供的命令行工具
  2. 使用 windows 终端，执行 MySQL [-h 127.0.0.1] [-p 3306] -u root -p，需要提前设置环境变量

数据模型

客户端发送 SQL 语句给服务端，服务端中的 DBMS 负责操作管理数据库，每个数据库中都有一个或多个表，数据存储在表中，DBMS 解析SQL 语句，并执行相应的操作。
