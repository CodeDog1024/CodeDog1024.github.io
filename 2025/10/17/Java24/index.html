File 类

File 类是 Java 中最常用的类之一，用于操作文件。根据文件路径来创建 File 对象。然后通过 File 对象，来获取文件信息，或者操作文件。File 其实也可以获取文件夹，毕竟文件夹也可以看做是文件。

File 类的构造方法

构造方法 描述

File(String parent, String child) 根据父目录路径和文件名创建 File 对象
File(File parent, String child) 根据父目录 File 对象和文件名创建 File 对象
