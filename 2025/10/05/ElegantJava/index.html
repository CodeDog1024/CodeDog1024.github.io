卫语句

卫语句（Guard Clause）是一种编程技巧，用于提前处理特殊情况或异常条件，从而避免深层嵌套的 if-else 结构。

取最值

Records

Records 是 Java 14 新增的语法，用于创建数据类。数据类是一种特殊的类，它继承自 Object 类，并实现了 Serializable 接口。数据类提供了一些默认的构造函数和 getter 方法，以及 equals、hashCode 和 toString 方法。

Stream

Stream 是 Java 8 新增的流式处理 API，它允许用户在一系列元素上执行一系列操作，例如过滤、映射、聚合等。

Optional

Optional 是 Java 8 新增的容器类，用于表示可能为空的值。它提供了一些有用的方法，例如isPresent、ifPresent、orElse等。

Try-with-resources

try-with-resources 是 Java 7 新增的语法，用于确保资源在 try 块结束后被正确关闭。

Lambda

Lambda 是 Java 8 新增的语法，用于创建匿名函数。Lambda 语法允许用户使用更简洁的语法来定义匿名函数。
