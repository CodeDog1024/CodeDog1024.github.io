HTML5 语义结构

发表于2025-12-21|更新于2025-12-22
|浏览量:

开始学习 HTML

超文本标记语言（HyperText Markup Language）的英文首字母缩写，是一种用来告知浏览器如何组织页面的标记语言。

  1. 标记可以理解为元素标签，如：h1 元素，语法为 <h1></h1>，使用 <> 包裹，第一个为开始标签，第二个为结束标签。
  2. 元素标签不区分大小写，即：<h1> 和 <H1> 是一样的，但是推荐小写。
  3. 一个 HTML 是由一系列的元素标签组成的。

标签语法

标签按结构可分为：双标签和单标签，如 <h1></h1> 这种由开始标签和结束标签组成的称为双标签，而像 <hr> 这种称为单标签。

HTML 文档

我们在使用编辑器创建 HTML 文件时，会自动生成一个 HTML 文档，其中包含了一些标签。

标签关系

在编写元素标签时，根据各元素标签之间位置不同，可分为两种关系：并列关系（兄弟）和嵌套关系（父子）。

并列关系

在 HTML 文档中，我们就可以发现 head 元素和 body 元素之间是并列关系。

嵌套关系

同样 head 元素和 mate 元素或者 title 元素之间是嵌套关系。

标题和段落标签

标题元素为：h1、h2、h3、h4、h5、h6，语法为：<h1>标题</h1>，h1 为一级标题，程度依次递减，h6 最小。

  1. 在页面显示时，标题元素独占一行。
  2. 在一个页面中，建议 h1 元素只存在一个，且标题层次嵌套不要超过 3 层

段落元素为：p，语法为：<p>文本</p>。

  1. 段落元素中的段落文本会根据显示宽度自动换行。

强调与重要性标签

HTML 中，提供了相应的标签，来使文本具有加粗、倾斜、下划线等效果，以此来突出文本的强调和重要性。

标签 描述
strong 加粗
em 斜体
ins 下划线
del 删除线

注释标签

HTML 中，提供了注释标签，用于对代码进行注释，注释标签的语法为：<!-- 注释内容 -->。

  1. 注释标签以及其内容不会被浏览器显示。
  2. 注释标签中的内容可以换行，但是注释标签不能嵌套。

快级标签和内联标签

标签的分类出了双标签和单标签，还可以根据标签的展示形式分为：块级标签和内联标签。

块级标签：

  1. 块级标签会独占一行。
  2. 块级元素可以内嵌其他元素。

内联标签：

  1. 内联标签不会独占一行，如果一行内足够放，则内联标签会在一行内水平排列。
  2. 内联元素不可以内嵌块级元素，但可以内嵌内联元素。

注意：

  1. 相同标签不能互相嵌套。
  2. 标签的展示形式不是固定的，可以通过 CSS 来改变。

图片标签

图片元素为：img，语法为：<img src="图片地址" alt="图片描述">，这是一个单标签，包含了两个属性：src 和 alt，一个用来声明图片地址，一个用来描述图片。
