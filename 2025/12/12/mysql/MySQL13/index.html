系统数据库

在 MySQL 安装后，我们可以发现 MySQL 中就已经存在了四个数据库，分别是：information_schema、mysql、performance_schema、sys。

数据库 用途
mysql 存储 MySQL 服务器正常运行时所需要的各种信息（时区、主从、用户、权限等）
information_schema 提供了方法问数据库元数据的各种表和视图，包含数据库、表、字段类型及访问权限等
performance_schema 为 MySQL 服务器运行时状态提供了一个底层监控功能，用于手机数据库服务器性能参数
sys 包含了一系列方便 DBA 和 开发人员利用 performance_schema 性能数据库进行性能调优和诊断的视图

常用工具

mysql

mysql 的客户端工具

连接数据库语法：mysql [options] [database]

mysqladmin

mysqladmin 是一个执行管理操作的客户端程序，可以用她来检查服务器的配置和当前状态，创建并删除数据库等。

mysqlbinlog

由于服务器生成的日志文件都是以二进制格式保存的，所以如果想要检查这些文件，就需要用到 mysqlbinlog 日志管理工具

mysqlshow

mysqlshow 客户端对象查找工具，可以很快的查找存在哪些数据库、数据库中的表、表中的列和索引等。

mysqldump

mysqldump 客户端工具主要用来备份数据库或者在数据库之间进行数据迁移。

mysqlimport/source

mysqlimport 是客户端数据导入工具，前边提到 mysqldump 使用 -T 指令可以生成表结构文件和数据文件两个问题，mysqlimport 就是用来将数据文件导入到数据库中的。

总结

  1. mysql：客户端连接工具
  2. mysqladmin：MySQL 管理工具
  3. mysqlbinlog：二进制日志文件查看工具
  4. mysqlshow：数据库、表、字段信息查看工具
  5. mysqldump：数据库备份工具
  6. mysqlimport：数据导入工具
