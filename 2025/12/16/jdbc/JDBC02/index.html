JDBC 获取数据库连接

发表于2025-12-16|更新于2025-12-18
|浏览量:

加载驱动

  1. 下载驱动文件 Connector/J 9.5.0
  2. 解压文件，找到 jar 文件
  3. 在工程根目录下创建 lib 文件夹，并添加 jar 文件
  4. 将 lib 文件夹添加到工程库

连接信息

  1. URL：jdbc:mysql://IP 地址:端口号/数据库名
  2. Properties：用户名、密码

连接方式

连接方式一

最基础的连接方式

连接方式二

在连接方式一的基础上，修改驱动加载方式，采用反射而不是硬编码，使程序具有更好的可移植性。

连接方式三

在连接方式二的基础上，引入 DriverManager 替换 Driver。

连接方式四

在连接方式三的基础上删除部分代码，因为注册驱动的操作 Java 在加载驱动时会帮我们做。

连接方式五（最终版）

在连接方式四的基础上，将所有的配置信息写入配置文件中，而不是硬编码的方式写在代码中，实现了数据和代码的分离。
