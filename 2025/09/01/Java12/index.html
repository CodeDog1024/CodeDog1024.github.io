集合的基本使用

为什么要有集合

当我们需要存储多个同一类型的数据时，我们可以使用数组，但是数组的长度是固定的，无法动态调整。而集合可以根据需要动态扩展，提供了更灵活的数据存储方式。但是相比及数组，集合无法存储基本数据类型的数据。

ArrayList

ArrayList是 Java 中最常用的集合类之一，它实现了List接口，允许存储重复的元素，并且元素有序。

subList(int fromIndex, int toIndex) 返回列表中指定范围的子列表
sort(Comparator<? super E> c) 对列表中的元素进行排序

包装类

前边提到，集合无法存储基本数据类型的数据，因此，包装类诞生了，包装类其实就是基本数据类型所对应的引用数据类型的形式。如：byte -> Byte。
