为什么要使用变量

变量：在程序运行时，用来存储数据，且变量中存储的数据可以被改变。

变量的介绍

概念

变量是程序设计中的基本概念，它代表了内存中的一块存储空间，变量用于保存程序运行过程中需要处理的数据，变量中的数据可以在程序执行过程中被修改。

使用的基本步骤

  1. 变量的声明：数据类型 变量名
  2. 变量的赋值: 变量名 = 值
  3. 变量的使用: 变量名

变量使用注意事项

  1. 变量表示内存中的一块区域，用于存储数据。根据存储数据的不同，变量的数据类型也不同。存储数据的区域大小也不同
  2. 声明变量时，必须指定变量的数据类型
  3. 在使用变量之前，必须先声明变量，且变量必须赋值
  4. 变量内存储的数据是可以被修改的，但是类型不能被修改
  5. 变量在同一作用域下不可重名

+ 号的使用

规则

  1. 对于数值类型，+号表示加法运算
  2. 如果有字符串参与运算，+号表示字符串的拼接

数据类型分类

存储不同的数据，使用不同的数据类型，数据类型决定了变量的取值范围和存储空间大小。

基本数据类型

不同的数据类型在内存中所占的存储空间大小不同，byte 占 1 字节，1 字节 = 8 位，因此 byte 的取值范围是 -2^7 ~ 2^7-1，即 -128 ~ 127。取值范围的计算涉及二进制知识，这里不再赘述。

short 2 16 -2^15 ~ 2^15-1
int 4 32 -2^31 ~ 2^31-1
long 8 64 -2^63 ~ 2^63-1
float 4 32 1.4E-45 ~ 3.4E+38
double 8 64 4.9E-324 ~ 1.8E+308

整数类型

整数类型分类

整数类型注意事项

  1. 与 C 语言不同，在任何平台下，Java 中的基本数据类型所占的字节大小都是固定的
  2. 在定义整数时，默认为 int 类型，如果需要定义 long 类型，需要在数字后面加上 L 或 l，建议使用大写的 L，避免与数字 1 混淆
  3. 在取值范围能够满足使用的情况下，尽量使用取值范围小的数据类型，节省内存空间
  4. bit 是计算机中最小的存储单位，8 bit = 1 byte

浮点类型

浮点类型分类

数据类型 字节 位宽 取值范围
float 4 32 1.4E-45 ~ 3.4E+38
double 8 64 4.9E-324 ~ 1.8E+308

浮点类型注意事项

  1. 浮点数在计算机中存储时，由符号位、阶码、尾数三部分组成
  2. 由于计算机存储时精度问题，尾数部分可能丢失，造成误差
  3. 在定义浮点数时，默认为 double 类型，如果需定义 float 类型，则需要在小数后面加上 F 或 f，建议使用 double，精度更高，误差更小
  4. 浮点数在运算时会丢失精度，尽量避免使用浮点数进行精确运算，且不要使用运算结果进行相等判断，如：double num = 0.1 + 0.2;，此时 num 的值为 0.30000000000000004，所以如果进行 num 是否等于 0.3 的判断，结果为不相等，会引发业务逻辑上的 bug
