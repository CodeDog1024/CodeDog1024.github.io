注释

为什么要有注释

注释就是对代码进行解释说明的文字，通过注释可以快速了解一个程序的功能和实现思路。

注释分类

  1. 单行注释：// 注释内容。
  2. 多行注释：/* 注释内容 */。
  3. 文档注释：/** 注释内容 */。

代码示例

注意事项

  1. 注释不会影响程序的运行，编译器会自动忽略注释内容。若代码被注释，则该代码不会被执行。
  2. 多行注释不可以进行嵌套。

关键字

什么是关键字

关键字就是在 Java 中被赋予了特殊含义的单词，如：class、public、for 等。

关键字 class

class：用于创建/定义一个类。被 class 修饰的名字称为类名

字面量

什么是字面量

字面量是指在代码中直接表示的固定值，如数字、字符、字符串等。例如：100、'A'、"Hello"。

字面量分类

类型 说明 举例

小数 小数 3.1415926
字符 被单引号包括的一个字符 ‘A’
字符串 被双引号包括的字符 “ABC”
布尔 只有两个值：true 和 false true
空 没有值 null

变量-基本用法

什么是变量

变量即在程序运行过程中，值会发生变化的量。变量是用来存储数据的容器，可以在程序中随时读取和修改。

变量的定义格式

数据类型 变量名 = 值;

变量-使用方式和注意事项

使用方式

  • 若想使用变量中的值，直接使用变量名即可。如：System.out.println(age); // 10。
  • 变量可以参与计算，如：System.out.println(age + 2); // 20。
  • 修改变量的值只需要对变量重新赋值即可，如：age = 30;，不能重复定义。

代码示例

注意事项

  1. 一个变量只能存储一个值
  2. 变量在同一作用域内不能重复定义
  3. 一条语句可以定义多个变量
  4. 变量在使用前必须赋值

计算机中的数据存储

在计算机中，任意数据都是以二进制的形式存储的。

进制

进制 说明 值 书写格式
十进制 满十进一 0 ~ 9 无
二进制 满二进一 0、1 0b 开头
八进制 满八进一 0 ~ 7 0 开头
十六进制 满十六进一 0 ~ 9、A ~ F 0x 开头

进制之间的转换

其他进制转十进制

  • 二进制转十进制：从右到左，依次乘以 2 的权次幂，然后求和。
  • 八进制转十进制：从右到左，依次乘以 8 的权次幂，然后求和。
  • 十六进制转十进制：从右到左，依次乘以 16 的权次幂，然后求和。

十进制转其他进制

  • 十进制转二进制：不断地将十进制数除以 2，记录下余数，直到商为 0，然后将余数倒序排列。
  • 十进制转八进制：不断地将十进制数除以 8，记录下余数，直到商为 0，然后将余数倒序排列。
  • 十进制转十六进制：不断地将十进制数除以 16，记录下余数，直到商为 0，然后将余数倒序排列。

ASCII 码表

  • 已知计算机中的数据都是以二进制的形式存储，那么对于字符来说，是如何转为二进制的呢？这里就需要用到 ASCII 码表，其中记录了字符与数值之间的关系，通过将字符转为数值再进行存储，如：a 在 ASCII 码表中对应的十进制为 97，二进制为 0110 0001。
  • ASCII 码表中存储的字符数量很少，其中就缺少中文汉字。因此，中国便自己推出了 GB2312 码表，其中便记录了汉字与数值之间的关系。而后，台湾又推出了 BIG5 码表，记录了繁体中文。
  • GBK 编码：收录了全部的中日韩汉字。
  • Unicode 编码：收录了世界各种语言的字符。

数据类型

数据类型的分类

基本数据类型

数据类型 关键字 字节(byte) 位(bit) 取值范围
整数 byte 1 8 -2^(8-1) ~ 2^(8-1)-1
整数 short 2 16 -2^(16-1) ~ 2^(16-1)-1
整数 int 4 32 -2^(32-1) ~ 2^(32-1)-1
整数 long 8 64 -2^(64-1) ~ 2^(64-1)-1
浮点数 float 4 32 -3.401298e-38 ~ 3.402823e+38
浮点数 double 8 64 -4.9000000e-324 ~ 1.797693e+308
字符 char 2 16 0 ~ 2^16-1
布尔 boolean 1 8 true、false

注意事项

  1. 在定义整数时，默认是 int 类型，而定义小数时，默认是 double 类型。
  2. 在定义 long 类型的变量时，需要在值后边追加 L，如：long num = 123456L;。
  3. 在定义 float 类型的变量时，需要在值后边追加 F，如：float num = 3.14F;。
  4. 根据字节大小，按照取值范围排序：double > float > long > int > short > byte

标识符

什么是标识符

标识符是程序中用来表示变量、函数、类等的名称。

命名规则

  1. 由字母、数字、下划线(_)和美元符号($)组成，但不能以数字开头。
  2. 区分大小写。
  3. 不能使用 Java 的关键字和保留字。
  4. 变量名或类名常采用小驼峰命名法，如：studentName、getStudentName。
  5. 类名常采用大驼峰命名法，如：Student、GirlFriend。
  6. 命名时应见名知其意，如：age = 18、name = "张三"，而不是 a = 18、b = "张三"。

键盘录入

使用Scanner类进行键盘录入。

使用步骤

  1. 导包：import java.util.Scanner;。
  2. 创建Scanner对象：Scanner scanner = new Scanner(System.in);。
  3. 读取输入：String input = scanner.nextLine();。
  4. 关闭Scanner：scanner.close();。
