API 和 API 文档

API 就是一个接口，接口就是一组方法，方法就是一组功能。通过阅读 API 文档，可以了解 API 的功能，以及如何使用 API。

String 概述

String 是引用数据类型，所以 String 对象需要通过 new 关键字创建，如：String str = new String("hello world");，但是我们之前都是 String str = "hello world"; 创建的，是因为 java 对此做了优化。String 对象的值是不可修改的，即便使用 str = "abc";，也只是创建了一个新的字符串覆盖了原来的字符串。

构造方法 说明
String() 创建一个空字符串
String(String original) 创建一个字符串，该字符串包含字符串 original 的内容
String(char[] value) 创建一个字符串，该字符串包含字符数组 value 的内容
String(byte[] value) 创建一个字符串，该字符串包含字节数组 value 的内容

直接赋值

当我们创建字符串时，JVM 会先检查字符串常量池中是否已经存在相同内容的字符串，如果存在，则直接返回该字符串的引用；如果不存在，则在堆中创建一个新的字符串对象，并将其引用放入常量池中。

new 关键字

只要是通过 new 关键字创建的，就会在堆中创建一个新的字符串

== 和 equals

对于基本数据类型，== 比较的是真实的值，而对于引用数据类型，== 比较的是地址值。所以 String 类提供了 equals 和 equalsIgnoreCase 方法来比较两个字符串的内容是否一致。
