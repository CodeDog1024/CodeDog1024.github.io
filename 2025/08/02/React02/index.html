表单受控绑定

  • 使用 React 中的 useState 来管理表单输入的状态。

React 中获取 DOM

  • 使用 useRef 来获取 DOM 元素的引用，先使用 useRef 创建一个引用对象，然后使用 ref 属性将其绑定到需要获取的 DOM 元素上。然后通过 current 属性访问该 DOM 元素。

处理 id 和时间

id 可以使用 uuid 库生成

  • 安装 npm install uuid

时间可以使用 dayjs 库处理

  • 安装 npm install dayjs

清空内容和聚焦实现

  • 清空内容可以使用 useRef 来获取输入框的引用，然后调用 current.value = '' 来清空内容或者直接使用 setComment('') 来清空状态。聚焦可以使用 inputRef.current.focus() 来实现。

组件通信-父传子-基础实现

  • 组件通信就是在 React 中不同组件之间传递数据和事件。
  • 当一个组件内嵌套了另一个组件时，此时该组件称为父组件，内嵌的组件称为子组件。
  • 父组件可以通过 props 将数据和事件传递给子组件，子组件可以通过 props 接收父组件传递的数据和事件。

组件通信-父传子-props 说明

  1. props 可传递任意类型的数据
  2. props 是只读的，其中的数据不能被修改

组件通信-父传子-children

  • 父组件在给子组件传递参数时，若将参数编写在子组件的标签内，则该参数会被自动传递给子组件的 props.children 属性。
  • children 是一个数组，包含了子组件标签内的所有内容。
