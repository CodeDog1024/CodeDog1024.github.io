什么是方法

方法是指一组执行特定任务的代码块，可以被重复调用。它通常包含方法名、参数列表和返回值类型。通过定义方法，可以提高代码的复用性和可维护性。方法是程序中最小的执行单元。

参数

在定义方法时，需要外部提供一些数据，才能完成程序的运行，这些数据称为方法的参数(形参)。调用者在调用方法时，需要提供参数(实参)。如果参数的类型是基本数据类型，那么就是将实参的值赋给形参，当形参的值发生改变时，实参不受影响。如果参数的类型是引用数据类型，那么就是将实参的地址赋给形参，当形参的值发生改变时，实参也会改变。

方法的小结

  1. 方法的执行需要调用，调用方法时需要使用方法名，并给出参数值。且参数的类型和个数必须与方法定义一致。
  2. 方法可以嵌套调用，但不能嵌套定义。
  3. 即使方法的返回值类型是 void，仍可以使用 return 语句返回，但是 return 后面不能跟任何内容。可以用于结束方法的执行，

方法的重载

什么是方法重载

方法重载（Overload）是指同一个类中，方法名相同，而参数列表不同的多个方法。参数列表不同包括：个数不同、类型不同、顺序不同。

方法的基本内存原理

函数调用会创建一个栈帧，栈帧中保存了函数的参数、局部变量、返回地址等信息。当函数返回时，栈帧会被销毁。当方法进行嵌套调用时，栈帧会按照调用顺序依次创建和销毁。

基本数据类型和引用数据类型

分类

  • 基本数据类型：int、float、double、char、boolean、long、short、byte。
  • 应用数据类型：需要使用 new 关键字创建的变量。

区别

  • 基本数据类型：存储的是数据本身。
  • 应用数据类型：存储的是数据所在的地址，真实的数据存储在堆内存中。

方法的值传递

分类

  • 值传递：方法参数是基本数据类型，传递的是参数值。
  • 引用传递：方法参数是应用数据类型，传递的是参数地址。
