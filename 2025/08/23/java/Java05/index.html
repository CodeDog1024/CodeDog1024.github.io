无限循环

无限循环，即无循环条件或者循环条件一直为 true 时，就会进入无限循环，也称死循环。

跳转控制语句

在循环中，可以使用跳转控制语句来控制循环的执行。

  • continue：跳过当前循环的剩余部分，直接进入下一次循环。
  • break：跳过当前循环的剩余部分，结束全部循环。
