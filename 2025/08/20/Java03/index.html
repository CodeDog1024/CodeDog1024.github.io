运算符

运算符和表达式

  • 运算符就是用于对数据进行操作的符号，如：+、-、*、/ 等。
  • 表达式是由运算符和操作数组合而成的，如：a + b、c * d 等。

算数运算符详解和综合练习

算数运算符分类

符号 作用 说明
+ 相加
- 相减
* 相乘
/ 相除 对于整数来说，结果为商，对于小数来说，才会整除
% 取模 当负数取余时，结果的符号同运算符左变量的符号相同

隐式转换和强制转换

拓展：“+” 的用法

数字相加

  • 两数相加时，若数据类型不一致，需要先统一数据类型，这就需要类型转换。

类型转换的分类

  1. 隐式转换：取值范围小的数据类型转为取值范围大的数据类型。
  2. 强制转换：取值范围大的数据类型转为取值范围小的数据类型。
  3. byte < short < int < long < float < double。
  4. byte、short、char 这三种数据类型在计算时，先转为 int 类型，再做计算。

隐式转换

强制转换

  • 格式：(目标数据类型) 原数据，如：int a = (int)10.0;。
  • 当原数据类型的值超出转换后数据类型的取值范围时，数据会发生错误。

字符串相加

  • 当有字符串参与了加法的执行，那么就会变成拼接而不是相加，如："123" + 123 = "123123"。

数值拆分优化

字符相加

  • 已知，char 类型在参与计算式，会优先转换为 int 类型，且前边学习了 ASCII 码表，记录了字符与数值之间的对应关系。所以当有字符参与计算时，实际上是以其 ASCII 码值进行计算的。如：'a' + 'b'，其实就是 97 + 98。
