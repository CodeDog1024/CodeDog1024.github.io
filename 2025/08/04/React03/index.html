Redux 快速上手

  • Redux 是一个 JavaScript 状态管理库，常与 React 一起使用。类似于 Vue 中的 Vuex 和 Pinia

使用步骤

  1. 定义一个 reducer 函数，处理状态变更
  2. 使用 createStore 方法，传入 reducer 函数，创建 Redux store 对象
  3. 使用 store.subscribe 方法，监听状态变更
  4. 使用 store.dispatch 方法，触发状态变更
  5. 使用 strore.getState 方法，获取当前状态
  • 引入 redux

Redux 与 React -环境准备

  1. 安装 Redux-Toolkit，这是 Redux 官方推荐的工具包，简化了 Redux 的使用。
  2. 安装 React-Redux，这是 React 官方推荐的 Redux 绑定库。
  • 文件目录

Redux 与 React -实现 counter

  • 创建 store/modules/counterSlice.ts 文件
  • 创建 store/index.ts 文件
  • 在 src/main.tsx 文件中，使用 Provider 组件，将 Redux store 对象传递给 React 组件
  • 在 src/App.tsx 文件中，使用 useSelector 和 useDispatch 钩子，获取状态和分发 action

Redux 与 React -提交 action 传参

  • 在 store/modules/counterSlice.ts 文件中，新增带参数的 incrementByAmount action
  • 在 src/App.tsx 文件中，使用 incrementByAmount action 并传入参数
