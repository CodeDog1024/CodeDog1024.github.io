JSX 基础-概念和本质

什么是 JSX

  • JSX 是一种 JavaScript 的语法扩展，允许我们在 JavaScript 代码中编写类似 HTML 的语法。

JSX 的本质

  • JSX 并不是标准的 JavaScript 语法，而是基于 JavaScript 的语法扩展，浏览器本身并不能识别 JSX 语法，需要通过编译工具（如 Babel）将 JSX 转换为标准的 JavaScript 代码，再进行执行。

JSX 基础-识别 JS 表达式

  • 在 JSX 中，可以使用大括号 {} 来嵌入 JavaScript 表达式。这些表达式可以是变量、函数调用或任何有效的 JavaScript 代码。

JSX 基础-实现列表渲染

  • 在 JSX 中，可以使用 JavaScript 的数组方法（如 map）来渲染列表。通过遍历数组，可以生成一组 JSX 元素。
  • 在渲染列表时，需要为每个元素提供一个唯一的 key 属性，以帮助 React 识别哪些元素发生了变化、添加或删除。

JSX 基础-实现基础条件渲染

  • 在 JSX 中，可以使用 JavaScript 的条件语句（如&&、三元运算符等）来实现条件渲染。根据不同的条件，可以选择性地渲染某些元素。

JSX基础-实现复杂条件渲染

  • 在 JSX 中，可以使用函数来实现更复杂的条件渲染逻辑。通过将条件判断封装在函数中，可以使代码更加清晰和可维护。
  • 也可以在 JSX 中直接插入匿名函数来进行条件渲染，但是不推荐使用，及影响效率又影响代码阅读

React 中的事件绑定

  • 在 JSX 中，可以使用 on + 事件 = {函数} 的格式来绑定事件处理函数。事件处理函数可以是一个普通的 JavaScript 函数，也可以是一个箭头函数。
