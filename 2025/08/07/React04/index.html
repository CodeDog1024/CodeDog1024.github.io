ReactRouter -快速开始

  • 安装命令 npm install react-router-dom 或者 npm install react-router
  • 浏览器环境推荐使用 react-router-dom

ReactRouter -路由导航跳转

声明式导航

  • 使用 <Link to="路径" /> 来实现路由导航，会被渲染成 <a href="路径" />

编程式导航

  • 使用 useNavigate 钩子函数获取一个 navigate 方法，传入 路径 即可

ReactRouter -导航跳转传参

  1. 传递时，将参数拼接在路径后面，如：/article?id=1&name=tom，然后使用 useSearchParams 获取参数
  1. Rest 风格，即在传递参数时，直接将值写在在路径中（需要提前在路径中配置占位符），无需写成 key=value的形式，如：/article/1/tom，然后使用 useParams 获取参数

ReactRouter -嵌套路由配置

  1. 使用 children 属性配置子路由
  2. 使用 Outlet 组件来渲染子路由

ReactRouter -默认二级路由配置

  • 将二级路由的 path 属性移除，并添加 index: true

ReactRouter -404 路由配置

  • 使用 path: '*' 来配置 404 路由

ReactRouter -两种路由模式

路由模式 url 底层逻辑 是否需要后端支持 创建方法
history url/login histort + pushState 需要 createBrowserRouter
hash url/#/login hashChange 不需要 createHashRouter
memory url/ memory 不需要 createMemoryRouter

数据 Mock 实现

什么是 Mock

  • 前后端进行分离开发时，前后端开发进度不一致，当后端无法提供对应的 API 接口时，前端可以通过 Mock 数据来模拟请求数据。

操作步骤

  1. 安装依赖包：npm i json-server
  2. 创建数据文件：src/server/db.json
  3. 启动服务器：json-server --watch db.json
