了解异常

什么是异常

  • 异常是程序运行过程中出现的错误，程序会因此停止运行，就是所谓的 BUG
  • 比如当我们想读取一个不存在的文件时，就会提示 FileNotFoundError 这就是异常

异常的捕获方法

为什么要捕获异常

  • 世界上没有完美的程序，为了保证程序在发生错误的同时依旧可以继续运行，所以我们需要尽可能的处理能够预知，且可以避免的错误，我们称之为捕获异常

捕获异常

  • 使用 try...except... 语句，将可能会发生异常的代码包含在 try 语句中，当运行时发生异常，except 语句会捕获这个异常，并执行 except 语句中的代码
  • 当我们能够预知异常的类型的时候，我们可以使用 except 异常类型 as e: 来捕获指定异常，比如读取不存在时，我们可以使用 except FileNotFoundError as e: 来捕获这个异常，再或者使用未定义的变量，使用 except NameError as e: 来捕获这个异常。
  • 但是这样子就不能捕获其他类型的异常了
  • 当需要捕获多个异常的时候，可以使用写在一个括号中，并使用逗号隔开
  • 捕获所有异常，使用Exception，其中Exception是所有异常的基类

else 语句

  • try 块没有异常发生时，会执行else块

finally 语句

  • try 块不管有没有异常发生，都会执行finally块

异常的传递性

  • 函数中如果发生异常，会向上传递，知道被捕获或者程序终止
  • 只需要在整个函数的调用过程当中，任意一个位置捕获异常，就可以解决

Python 模块

什么是模块

  • Python 中的模块，就是 Python 文件，其中包括编写好的变量和函数，其他文件可通过导入模块来使用该文件中的代码

模块的导入

  • Python 模块的导入方式：[from 模块名] import [模块 | 类 | 变量 | 函数 | *] [as 别名]
  • import 导入模块，需要使用 模块名 来调用模块中的函数、类、变量等。
  • 导入部分模块内的指定功能，或者使用 * 来导入所有功能。
  • 使用 as 语法为模块或者模块中的函数取别名
