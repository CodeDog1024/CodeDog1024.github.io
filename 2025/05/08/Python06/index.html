数据容器入门

为什么学习数据容器

  • 当我们需要存储多个数据时，使用数据容器可以更方便地管理和操作这些数据。
  • 数据容器可以帮助我们组织和处理数据，使得代码更加简洁和易读。
  • 数据容器提供了丰富的操作方法，可以方便地进行数据的增删改查等操作。

数据容器

  • 可以存储多个数据的数据类型
数据容器的分类 数据容器的特点
List（列表） 可变、可重复，支持增删改查操作
Tuple（元组） 不可变、可重复，适合存储不可变数据
Str（字符串） 不可变、不可重复，存储字符序列
Set（集合） 可变、不可重复，支持集合运算
Dict（字典、映射） 可变、不可重复，存储键值对

数据容器：List（列表）

列表的定义

  • 数据的类型可以是任意类型，当然也可以是列表

列表的下标索引

  • 可以通过列表的下标索引来访问列表中的元素，从左向右，从 0 开始，所以若想访问第二个元素，则需要使用下标 1
  • 也可以通过反向的下标索引来访问列表中的元素，从右向左，从 -1 开始

列表的常用操作

操作方法 说明
append() 追加元素
extend() 合并列表
insert() 插入元素
remove() 删除元素（通过值）
pop() 删除元素（通过下标）
del 删除元素（通过下标）
clear() 清空列表
count() 列表中指定元素个数
index() 获取元素下标
len() 列表长度
sort() 排序
reverse() 反转
copy() 列表拷贝

获取元素下标

  • index(value)：获取 value 在列表中的下标

修远指定下标的值

  • 直接赋值即可

插入元素

  • insert(index, value)：在下标 index 的位置插入元素 value

追加元素

  • append(value)：把元素 value 追加在列表末尾

合并列表

  • extend(list)：将 list 追加在列表尾部

删除元素（通过下标）

  • del list[index]：删除指定下标的元素
  • pop(index)：删除指定下标的元素，并把被删除的元素返回

删除元素（通过值）

  • remove(value)：删除第一个与指定值匹配的元素

清空列表

  • clear()：清空列表

列表中指定元素个数

  • count(value)：获取元素 value 的个数

排序

  • sort()：对列表进行排序，默认是升序
  • reverse()：反转列表中的元素

列表长度

  • len(list)：获取列表长度

数据容器：Tuple（元组）

  • 列表的元素是可以被修改的，而元组不可以修改，当需要被存储的数据不可被修改时，可以使用元组

掌握元组的定义格式

  • 定义元组使用小括号，同样使用逗号隔开，且元组中的元素可以是任何数据类型
  • 当元组内只有一个元素时，元组定义时，需要在元素后加上逗号，不然会被识别成一个变量
  • 元组同样支持嵌套

掌握元组的常见操作

操作 描述
元组[索引] 访问元组中的元素
index() 返回指定元素的索引
count() 返回指定元素的个数
len() 返回元组长度
in 判断元组中是否存在某个元素
  • 访问元组中的元素，同列表一样
  • 返回指定元素的索引
  • 返回指定元素的个数
  • 返回元组长度
  • 判断元组中是否存在某个元素
  • 遍历元组（while）
  • 遍历元组（for）

注意事项

  • 元组不持支修改，会报错
  • 若元组中嵌套了其他容器，且该容器支持修改，那么修改该容器内的元素，元组不会报错

  • 统计字符串中字符出现的次数
  • 字符串长度

数据容器：Set（集合）

  • 相比于之前学过的数据容器，集合中的元素是不可重复的，且是无序的，集合使用大括号{}来定义。
  • 因为集合是无序的，所以不能通过索引来获取集合中的元素。

  • 因为集合不支持索引，所以不能使用 while 循环进行遍历

数据容器：Dict（字典、映射）

  • 字典是一种键值对容器（key: value），键唯一，值可以重复，字典同集合一样，使用{}定义，只不过它的结构是 { key: value, key: value, ...}

字典的定义

  • 字典嵌套

字典的常用操作

操作 描述
my_dict[key] 获取字典中指定键的值
my_dict[key] = value 设置字典中指定键的值
del my_dict[key] 删除字典中指定键的值
key in my_dict 判断字典中是否存在指定键
key not in my_dict 判断字典中不存在指定键
get(key, default) 获取字典中指定键的值，如果键不存在则返回默认值
keys() 获取字典中所有键的列表
values() 获取字典中所有值的列表
items() 获取字典中所有键值对的列表
clear() 清空字典
pop(key, default) 删除字典中指定键的值，如果键不存在则返回默认值
popitem() 删除字典中最后一个键值对，并返回该键值对
update(other_dict) 更新字典，将另一个字典中的键值对添加到当前字典中
setdefault(key, default) 获取字典中指定键的值，如果键不存在则添加该键值对
clear() 清空字典
fromkeys(iterable, value=None) 使用给定的可迭代对象创建一个字典，每个键对应的值为给定的值
copy() 返回字典的浅拷贝
  • 访问字典元素，字典同样不支持下标访问，只能使用键访问
  • 新增/修改：若键不存在，则新增键值对，若键存在，则修改键值对
  • 删除：删除字典中指定键的值
  • 清空字典
  • 获取字典中全部的键值
  • 获取元素数量

数据容器总结

  • 支持下标索引
    1. 列表、元组、字符串
    2. 集合、字典
  • 支持元素重复
    1. 列表、元组、字符串
    2. 集合、字典
  • 支持元素修改
    1. 列表、集合、字典
    2. 元组、字符串
数据容器 元素数量 元素类型 下标索引 元素重复 元素修改 元素有序
列表 多个 任意 支持 支持 支持 是
元组 多个 任意 支持 支持 不支持 是
字符串 多个 字符 支持 支持 不支持 是
集合 多个 任意 不支持 不支持 支持 不是
字典 多个 任意 不支持 不支持 支持 不是

字符串比较大小

  • ASCII码表：每一个字符都有对应的ASCII码，ASCII码值越小，字符越小。比如，a的ASCII码是97，A的ASCII码是65，所以a比A小。
  • 字符串在比较时是按位比较，两个字符串分别取每一位的字符进行比较
