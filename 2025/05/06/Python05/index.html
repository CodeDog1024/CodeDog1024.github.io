函数介绍

函数

  • 函数：是指已完成编写，且可重复使用，具有特定功能的代码段

函数的定义

函数的定义

  • 注意：函数必须先定义后使用

案例

函数的参数

函数的传入参数

  • 在函数执行过程中，能够接收并使用外部提供的数据
  • 在定义函数时，声明的参数称为形参，而调用函数时传入的参数称为实参，在调用函数时，程序会将实参的值 copy 给形参，然后形参再去参与程序的运行，从而实现传参，因此，实参的个数和形参需一致

案例

函数的返回值

函数返回值的定义

什么是返回值

  • 返回值：函数执行完毕后，返回给调用者的结果，通过 return 语句实现。

返回值的语法

  • 在 return 之后的代码不会被执行，函数执行到 return 语句时，函数就会结束

None 类型

  • 当函数没有 return 语句时，或者 return 后没有返回值时，函数会默认返回 None，即空值

函数的说明文档

函数的说明文档

  • 在函数定义的第一行，可以使用三重引号 """ 来编写函数的说明文档，描述函数的功能、参数、返回值等信息，帮助调用者快速了解函数的作用已经调用方式

函数的嵌套调用

函数的嵌套调用

  • 在一个函数中调用另一个函数，称为函数的嵌套调用，且需要等待被调用的函数执行完毕后，才能继续执行当前函数

变量的作用域

局部变量

  • 在函数内部定义的变量，称为局部变量，只能在函数内部使用，函数外部无法访问
  • 局部变量随着函数的调用而创建，随着函数的结束而销毁，当函数执行完毕，局部变量也会消失

全局变量

  • 在函数外部定义的变量，称为全局变量，可以在函数内部和外部使用
  • 全局变量在程序运行时创建，直到程序结束才会销毁

global 关键字

  • 在函数内部对全局变量的值进行修改时，并不是真正的修改，而是创建了一个和全局变量同名的局部变量而已，所以在函数执行完后，会发现全局变量的值并没有被修改
  • 使用 global 关键字可以在函数内部声明全局变量，表示要使用全局变量，而不是创建一个同名的局部变量
