八、文件操作

发表于2025-05-19|更新于2025-05-20
|浏览量:

文件的编码

  • Python 中，文件编码默认为 UTF-8，如果文件编码为 GBK，则需要指定编码格式。

文件的读取

  • 使用 open() 函数打开文件，需要传入三个参数，分别为 文件名，打开模式，编码格式，编码格式默认为 UTF-8，返回值为文件对象。
  • 打开模式共分为三种，分别是 r：只读，w：只写，a：追加
读取方法 解释
read() 读取文件内容，可指定字节，返回字符串
readline() 读取一行内容，返回字符串
readlines() 读取所有行内容，返回列表
close() 关闭文件
with open() as fp 自动关闭文件
  • read() 方法
  • readlines() 方法
  • readline() 方法
  • with open() as fp 语法，会自动关闭文件

练习

文件的写入

  • 使用 write() 方法，写入内容，需要传入字符串，返回值为写入的字节数。
  • 当文件不存在时，使用 w 模式写入文件，会创建文件，并写入内容。
  • 写入文件后，需要调用 flush() 方法，刷新缓冲区，将内容写入文件，或者调用 close() 方法，关闭文件，也会将内容写入文件，或者使用 with open() 语法，自动关闭文件。

文件的追加

  • 使用 a 模式写入文件，追加内容，如果文件不存在，则创建文件。
  • 其他同文件的写入
