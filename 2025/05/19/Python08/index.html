八、文件操作

发表于2025-05-19|更新于2025-05-19
|浏览量:

文件的编码

  • Python 中，文件编码默认为 UTF-8，如果文件编码为 GBK，则需要指定编码格式。

文件的读取

  • 使用 open() 函数打开文件，需要传入三个参数，分别为 文件名，打开模式，编码格式，编码格式默认为 UTF-8，返回值为文件对象。
  • 打开模式共分为三种，分别是 r：只读，w：只写，a：追加
读取方法 解释
read() 读取文件内容，可指定字节，返回字符串
readline() 读取一行内容，返回字符串
readlines() 读取所有行内容，返回列表
close() 关闭文件
with open() as fp 自动关闭文件
  • read() 方法
  • readlines() 方法
  • readline() 方法
  • with open() as fp 语法，会自动关闭文件
