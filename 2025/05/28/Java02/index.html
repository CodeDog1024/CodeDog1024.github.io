发表于2025-05-28|更新于2025-06-02

注释

为什么要有注释

  • 注释就是对代码解释说明的文字，可以方便程序员阅读和理解代码

注释分类

  • 单行注释
  • 多行注释
  • 文档注释

注释的注意事项

  1. 被注释的代码不参与编译、运行
  2. 注释不能嵌套

关键字

什么关键字

  • 关键字：Java 语言中定义的、具有特殊含义的标识符

关键字：class

  • class：类，通过 class 类名 可以定义一个类

字面量

什么是字面量

  • 字面量：字面量是程序中直接给出的值，如：123、3.13、”hello”、true、false

字面量的类型

字面量类型 说明 举例
整数类型 无小数点的数字 123、-456
小数类型 有小数点的数字 3.14、-0.001
字符串类型 由双引号括起来的字符序列 “hello”、”Java”
字符类型 由单引号括起来的单个字符 ‘a’、’1’、’$’
布尔类型 表示真或假的值 true、false
空类型 表示没有值 null

转义字符

  • 转义字符：在字符串中使用的特殊字符，用于表示无法直接输入的字符或具有特殊含义的字符
转义字符 说明
\\ 反斜杠
\' 单引号
\" 双引号
\n 换行符
\t 制表符
\r 回车符
\b 退格符
\f 换页符
  • 制表符：使用空格将前面的字符串补齐到 8 个或者 8 的倍数，然后输出
  • 换行符：在字符串中使用 \n 可以实现换行
  • 回车符：在字符串中使用 \r 可以将光标移动到行首
  • 特殊字符：因为 \、'、" 等字符在 Java 中有特殊含义，所以不能直接在字符串中输出他们，需要使用 \ 进行转义

变量-基本用法

什么是变量

  • 变量：变量是程序中用于存储数据的命名空间，可以将指定类型的字面量存储到变量中，并且程序运行时其值可以被改变

变量的定义格式

  • 定义变量的格式为：数据类型 变量名 = 初始值;

变量-使用方式和注意事项

变量的使用

  • 访问变量的值
  • 参与计算
  • 修改变量的值

变量的注意事项

  • 在同一作用域内，变量名不能重复
  • 变量可以多个同时定义
  • 变量在使用前必须先定义并初始化
