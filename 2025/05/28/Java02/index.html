二、Java 基础概念

发表于2025-05-28|更新于2025-06-17
|浏览量:

注释

为什么要有注释

  • 注释就是对代码解释说明的文字，可以方便程序员阅读和理解代码

注释分类

  • 单行注释
  • 多行注释
  • 文档注释

注释的注意事项

  1. 被注释的代码不参与编译、运行
  2. 注释不能嵌套

关键字

什么关键字

  • 关键字：Java 语言中定义的、具有特殊含义的标识符

关键字：class

  • class：类，通过 class 类名 可以定义一个类

字面量

什么是字面量

  • 字面量：字面量是程序中直接给出的值，如：123、3.13、”hello”、true、false

字面量的类型

字面量类型 说明 举例
整数类型 无小数点的数字 123、-456
小数类型 有小数点的数字 3.14、-0.001
字符串类型 由双引号括起来的字符序列 “hello”、”Java”
字符类型 由单引号括起来的单个字符 ‘a’、’1’、’$’
布尔类型 表示真或假的值 true、false
空类型 表示没有值 null

转义字符

  • 转义字符：在字符串中使用的特殊字符，用于表示无法直接输入的字符或具有特殊含义的字符
转义字符 说明
\\ 反斜杠
\' 单引号
\" 双引号
\n 换行符
\t 制表符
\r 回车符
\b 退格符
\f 换页符
  • 制表符：使用空格将前面的字符串补齐到 8 个或者 8 的倍数，然后输出
  • 换行符：在字符串中使用 \n 可以实现换行
  • 回车符：在字符串中使用 \r 可以将光标移动到行首
  • 特殊字符：因为 \、'、" 等字符在 Java 中有特殊含义，所以不能直接在字符串中输出他们，需要使用 \ 进行转义

变量-基本用法

什么是变量

  • 变量：变量是程序中用于存储数据的命名空间，可以将指定类型的字面量存储到变量中，并且程序运行时其值可以被改变

变量的定义格式

  • 定义变量的格式为：数据类型 变量名 = 初始值;

变量-使用方式和注意事项

变量的使用

  • 访问变量的值
  • 参与计算
  • 修改变量的值

变量的注意事项

  • 在同一作用域内，变量名不能重复
  • 变量可以多个同时定义
  • 变量在使用前必须先定义并初始化

变量练习-计算公交车的人数

计算机中的数据存储

  • 在计算机中，任意数据都是以二进制的形式存储的

进制

  • 十进制：十进制是我们日常生活中最常用的计数系统，基数为 10，使用数字 0-9，满 10 进 1
  • 二进制：二进制是一种基于 2 的计数系统，基数为 2，使用数字 0 和 1，满 2 进 1
  • 八进制：八进制是一种基于 8 的计数系统，基数为 8，使用数字 0-7，满 8 进 1
  • 十六进制：十六进制是一种基于 16 的计数系统，基数为 16，使用数字 0-9 和字母 A-F，满 16 进 1

不同进制的表示方法

  • 十进制：直接使用数字表示，如 10、255
  • 二进制：使用 0 和 1 表示，在 Java 中以 0b 或 0B 开头，如 0b1010
  • 八进制：使用 0-7 表示，在 Java 中以 0 开头，如 0123
  • 十六进制：使用 0-9 和 A-F 表示，在 Java 中以 0x 或 0X 开头，如 0xA0

进制之间的转换

其他进制转十进制

  • 系数 * 基数 ^ 位数的和

十进制转其他进制

  • 循环除以基数，直到商为 0，取余数作为新进制的位数，逆序输出

字符在计算机中的存储

  • 字符无法直接存储在计算机中，而是通过 ASCII 或 Unicode 等编码方式转换为对应的数字，进而存储为二进制数据，比如: A 的 ASCII 码是 65，在计算机中存储为 01000001。
  • 不同的编码方式，所包含的字符集也不同，比如 ASCII 只能表示英文字符和一些特殊符号，GBK 则包含了汉字和繁体汉字，而 Unicode 可以表示世界上几乎所有的字符。

数据类型 关键字 大小 取值范围
整数类型 byte 1 字节 -128 ~ 127
整数类型 short 2 字节 -32768 ~ 32767

标识符

什么是标识符

  • 标识符是程序中用于唯一标识变量、方法、类、接口、属性、枚举等名称

命名规范

  • 标识符可以包含字母、数字、下划线、美元符
  • 不能以数字开头
  • 不能使用 Java 的关键字
  • 标识符区分大小写

命名建议

  • 命名尽量使用全小写，单词之间使用下划线连接或驼峰命名法，如 student_name 或 studentName
  • 见名知意，尽量使用有意义的名称，如 username、password，而不是 a、b 等无意义的名称
