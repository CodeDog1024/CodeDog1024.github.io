七、Python 函数进阶

发表于2025-05-18|更新于2025-05-19
|浏览量:

函数多返回值

  • 通过 return 一次只能返回一个变量，所以当该变量是一个列表或者元组时，便可以利用他们返回多个值

返回多个值

函数多种传参方式

位置参数

  • 函数调用时，按照参数的顺序依次传入参数值

关键字参数

  • 使用 key=value 的形式传入参数

不定长参数

  • 函数定义时，使用 *args 或 **kwargs 来定义一个不定长参数

缺省参数

  • 函数定义时，为参数设置默认值

匿名函数

函数作为参数传递

  • 函数作为参数传递给另一个函数，并让该函数执行不同的操作

lambda 函数

  • lambda 函数是 Python 中的一个匿名函数，它允许我们创建一个没有名称的函数，即匿名函数。
