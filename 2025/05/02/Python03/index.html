布尔类型和比较运算符

布尔类型

  • 布尔类型只有两个值：Ture（真）和Flase（假）

布尔类型的定义

  • 布尔类型为 bool，布尔类型的定义方法为：isOk = True 或 isOk = False

比较运算符

  • 布尔类型的变量也可以通过比较运算来获取，满足条件为 Ture，不满足条件为 False
运算符 描述 实例
== 判断内容是否相等 a = 3，b = 3，那么 a == b 的结果是 Ture
!= 判断内容是否不相等 a = 3，b = 4，那么 a != b 的结果是 Ture
> 判断左侧内容是否大于右侧内容 a = 4，b = 3，那么 a > b 的结果是 Ture
< 判断左侧内容是否小于右侧内容 a = 3，b = 4，那么 a < b 的结果是 Ture
>= 判断左侧内容是否大于等于右侧内容 a = 4，b = 3，那么 a >= b 的结果是 Ture
<= 判断左侧内容是否小于等于右侧内容 a = 3，b = 4，那么 a <= b 的结果是 Ture

if 语句的基本格式

if 判断语句

  • 当满足条件是，需要执行特定的代码

if else 语句

if else 组合语句

  • 当不满足条件时，也需要执行特定的代码

练习

if elif else 语句

  • 当条件过多时，且每个条件都有需要执行的代码，可以使用 if elif else 语句，条件从上至下依次判断，哪个条件成立就执行对应的代码，然后退出全部判断，若全部条件都不满足，执行 else 中的代码，当然 else 分支可以省略，但是当全部条件都不满足时，就不会有特定的代码去执行
