二、本地代码生成

发表于2025-05-13|更新于2025-05-13
|浏览量:

项目初始化

初始化根目录

  • 创建空项目 yuzi-generator

忽略无用提交

  • IDEA -> Settings -> Plugins
  • 搜索 .ignore 插件并安装，点击应用
  • 右键项目 -> New -> .ignore File -> .gitignore(Git)
  • 选中 Javva 和 JetBrains
  • 右键 .gitignore 文件 -> Git -> Add
  • 建议新建 README.md

创建 Demo 示例代码工程

  • 新建 yuzi-generator-demo-projects 目录，将 ACM 模板项目赋值到该目录下
  • 如果是自己手动建目录的话记得在项目结构里把 src 文件夹设为源代码，不然无法识别
  • MainTemplate.java

创建本地代码生成器项目

  • 创建 yuzi-generator-basic 项目
    1. 存放在 yuzi-generator 目录下
    2. 取消勾选 Create Git repository
    3. 使用 Maven 管理项目
    4. JDK 选择 1.8(我这里用的 21)
    5. 在高级设置中，指定 GroupId 和 ArtifactId
  • 创建好后要在 新窗口 打开项目
  • 在 pom.xml 文件中引入依赖
    1. Hutool
    2. Apache Commons Collections4
    3. Lombok
    4. JUnit4
  • 导入依赖后刷新 Maven
