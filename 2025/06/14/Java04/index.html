四、流程控制语句

发表于2025-06-14|更新于2025-06-17
|浏览量:

顺序结构

  • 从上至下，依次执行

分支结构

if

第一种格式（if）

  • 当关系表达式成立时，大括号内的代码才会执行，反之不执行，但是会继续执行之后的代码

第二种格式（if…else…）

  • 在第一种格式的基础上新增了 else 语句，当关系表达式不成立时，执行 else 大括号内的代码

第三种格式（if…else if…else…）

  • 第二种格式只考虑了两种情况，而第三种格式在其基础上新增了多种情况的可能，程序会一次判断每一个 if 中的关系表达式，如果哪个条件成立，就执行对应的代码，若均不满足，则执行末尾的 else 中的代码。

switch

格式

  • 依次比较表达式的结果与每个 case 对应的值受否相等，若相等，则执行对应的代码，代码执行完后，遇到 break; 语句，退出整个 switch，若无相等的值，则执行 default 的代码

注意事项

  1. default 可以省略不写，但是当所有 case 的值均不满足条件时，就无代码执行
  2. switch 中的 case 所对应的代码，在执行后只有遇到了 break 语句才会退出 switch，否则会继续执行下一个 case 所对应的代码，直到遇到 break 或者执行完全部的 case，称为case 穿透
  3. 在 JDK 12 之后，switch 支持箭头函数，可以省略 break 而不会造成穿透现象

循环结构

for

格式

  • 在条件判断语句成立的前提下，循环执行大括号内的代码直至条件判断语句不成立

练习

  • 求 1 ~ 5 的和
  • 求 1 ~ 100 中的偶数和

while

格式

for 和 while 区别

  • for 循环的初始化语句是在 for 循环内部，所以该变量只有 for 循环内部可以使用，而 while 循环是在外部，所以该变量在 while 外部也可以使用
  • for 循环中的条件控制语句是在执行代码后才会执行，而 while 中的条件控制语句并没有明确的执行顺序，完全取决于在编写代码时的编写位置，

练习

  • 折纸
  • 回文数
  • 除法

do…while

格式

  • 相比于 for 循环和 while 循环，do…while 循环不同的时在判断条件判断语句是否成立之前要先执行一次执行代码，所以，即使条件不成立，那么代码至少也会执行一次。
  • 即使条件不成立，也会执行一次代码
