什么是方法

什么是方法

  • 方法：是程序中最小的执行单元

方法的作用

  • 将程序中重复的代码进行封装，提高代码的复用性

方法的格式

方法的定义与调用

  • 无参
  • 有参
  • 带返回值

访问修饰符

  • 访问修饰符的级别从高到低依次为：public > protected > default > private
修饰符 作用
public 公有方法，可以在任何地方调用
protected 保护方法，可以在本包及其子类中调用
default 默认方法，只能本包中调用
private 私有方法，只能在本类中调用

静态修饰符

  • static：静态方法，不需要创建对象就可以调用，了解即可，后面会详解

返回值类型

  • 返回值类型，即返回值的数据类型，当方法需要有返回值时，返回值类型必须相同，若不需要返回值，则返回值类型为 void

参数列表

  • 在调用方法时，可以将一些必要的值传递给方法，作为参数，传递方称为实参，接收方称为形参
  • 在传递参数时，如果参数是基本数据类型，则传递的是值，形参改变，实参不会改变。如果是引用数据类型，则传递的是地址，形参改变，实参也会改变

方法的重载

什么是方法重载

  • 方法的重载：在同一个类中，存在多个同名方法，但是参数列表不同，称为方法的重载
  • 参数列表不同：个数不同、类型不同

练习

  • 遍历数组
  • 求最值
  • 存在判断
  • 数组复制

方法的内存

  • 方法保存在栈中，方法调用时，栈帧被创建，方法返回时，栈帧被销毁。

基本数据类型和引用数据类型

  • 基本数据类型：int, long, float, double, boolean, char, byte, short
  • 引用数据类型：上记以外
  • 基本数据类型是值类型，引用数据类型是引用类型。所以在调用方法时，基本数据类型会被复制，引用数据类型会被引用。基本数据类型的形参被改变时，实参不受影响，引用数据类型的形参被改变时，实参也会改变。
