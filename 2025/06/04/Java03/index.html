三、运算符

发表于2025-06-04|更新于2025-06-12
|浏览量:

算数运算符

分类

运算符 描述
+ 加法运算符
- 减法运算符
* 乘法运算符
/ 除法运算符（取商）
% 取模运算符（取余）
  • 整数
  • 小数

注意事项

  • 除法对于整数来说是取商，而当有小数参与运算时，就变成了真正的除法
  • 小数在进行运算时，由于计算机存储数据的精度的影响，会导致结果精度错误，如：0.1 + 0.2 = 0.30000000000000004

综合练习

  • 将一个三位数的各位数字分离出来

隐式转换

  • 在进行运算时，如果参与运算的两个数据类型不一致，Java 会自动将取值范围小的类型转换为取值范围大的类型。如：int + double 的结果是 double 类型。
  • 取值范围由小到大：byte < short < int < long < float < double
  • 注意：byte、short 和 char 在进行运算时会被提升为 int 类型。

强制转换

  • 在运算时，因功能需要，需要把取值范围大的数据类型转换为取值范围小的类型，需要通过强制类型转换，格式为：(数据类型)变量

扩展（“+” 的其他操作）

字符串相加

  • + 对于数值类型的数据可以进行相加操作，但是对于字符串类型则进行拼接操作，如："Hello" + "World" = "HelloWorld"、"年龄" + 18 = "年龄18"
  • 当多个值相加时，是左到右依次计算的，所以：1 + 2 + 3 + "ABC" = "6ABC" 而不是 "123ABC"

练习

字符相加

  • char 类型在计算式，会先提升到 int 类型，至于怎么提升，则是通过编码表来将字符转换成对应的十进制数值，比如 A 对应的就是 65，a 对应的则是 97，所以 'A' + 'a' = 162，1 + 'a' = 98

自增自减运算符

分类

运算符 描述
X++ 自增 1（先用后增）
++X 自增 1（先增后用）
X– 自减 1（先用后减）
–X 自减 1（先减后用）
  • ++、-- 符号可以写在变量的左侧或者右侧，若该运算单独一行，则两者并无任何区别，但是如果参与其他的表达式，则有明显区别，左侧为先加后用，即先自增 1 后再参与表达式，右侧为先用后加，即先使用原始值参与表达式，之后再自增 1

赋值运算符

分类

运算符 描述 说明
= 赋值 a = b ，将 b 的值赋给 a
+= 相加后赋值 a += b 等于 a = a + b
-= 相减后赋值 a -= b 等于 a = a - b
*= 相乘后赋值 a *= b 等于 a = a * b
/= 相除后赋值 a /= b 等于 a = a / b
%= 取余后赋值 a %= b 等于 a = a % b
  • +=、-=、*=、/=、%= 在运算时，若遇到数据类型的精度问题，会自动强制类型转换，如 short a = 10, b = 20; 那么 a += b 则相当于 a = (short)(a + b)

关系运算符

分类

运算符 描述
== 判断两者是否相等，成立为 true，反之为 false
!= 判断两者是否不相等，成立为 true，反之为 false
> 判断左侧是否大于右侧，成立为 true，反之为 false
>= 判断左侧是否大于等于右侧，成立为 true，反之为 false
< 判断左侧是否小于右侧，成立为 true，反之为 false
<= 判断左侧是否小于等于右侧，成立为 true，反之为 false

逻辑运算符

  • 在生活中，我们常用 5 < x < 15 来表示条件，但是在 Java 中，这样是不被允许的，需要将其分解成两个表达式 5 < x 和 x < 15，然后用 &（与） 将其连接得到5 < x & x < 15，其含义为x 大于 5 并且小于 15，这里的 &（与） 就是逻辑运算符，其中还包括|（或）、^（异或）等运算符

分类

运算符 描述 说明
& 逻辑与 表并且，两边都为真，则结果为正
| 逻辑或 表或者，两边都为假，则结果为假
^ 逻辑异或 两边相同为假，不同为真
！ 逻辑非 取反

短路逻辑运算符

分类

运算符 描述 说明
&& 短路与 同逻辑与，当左侧是 false 时，结果就是 false，右侧不会执行
|| 短路或 同逻辑或，当左侧时 true 时，结果就是 true，右侧不会执行

练习

位运算符

  • 学习位运算符之前需要先了解原码、反码和补码的相关知识。

原码、反码和补码

  • 原码就是十进制数值的二进制表示形式，最左侧的是符号位，0 为正，1 为负，如：3 的原码为0000 0011，-3 的原码为 1000 0011，其中每一位称为 1 bit，8 bit 称为 1 byte，即 8 位 = 1 字节，所以 1 字节的最大值为 0111 1111 即 127，最小值为 1111 1111 即 -127
  • 数据在计算的时候，并不是使用原码去参与计算的，而是使用补码去计算的，并且通过计算得到的结果也是补码
  • 正数：原码 —-> 反码 —-> 补码
  • 负数：原码 –(除符号位取反)–> 反码 –(加 1)–> 补码

分类

运算符 描述
& 按位与
|按位或
^ 按位异或
~ 按位取反
<< 左移
>> 右移（有符号）
>>> 右移（无符号）

三元运算符

  • 三元运算符其实可以理解为一种特殊的表达式，其格式为 关系表达式 ? 结果1 : 结果2，其作用是若关系表达式的结果为 true，则返回结果1，反之返回结果2

练习

instanceof 运算符

  • instanceof 运算符用于检查一个对象是否属于指定的类型，属于则返回 true，反之返回 false，目前了解即可，后边学到对象在做解释。
